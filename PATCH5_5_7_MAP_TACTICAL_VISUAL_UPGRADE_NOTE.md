# PATCH 5.5.7 — MapTacticalVisualUpgrade

## 기준

- 기준 버전: `MapPatch5_5_6_MapPerformanceMarkerFineTuneQA`
- 제외: `MapPatch5_5_6_1_MapPerformanceHotfix`

5.5.6.1은 지도 렉 체감이 더 커졌으므로 이번 정식 라인에서는 사용하지 않는다. 5.5.7은 5.5.6의 지도 구조를 기준으로 시각 레이어만 보수적으로 추가한다.

## 목표

지역지도를 단순 좌표 지도에서 U.A.C 전술 관제 화면처럼 보이게 개선한다.

## 포함 내용

### 1. 오염권 해칭 강화

- 레드존: 사선 해칭 + 외곽 위험면
- 블랙존: 심부 코어 + 어두운 해칭
- 옐로우존: 저밀도 감시 해칭
- 화이트/그린존: 낮은 명도 보호/관리권 표현

### 2. 감시 반경 / 위험 반경

- 오염권 중심부 주변 감시 반경
- 봉쇄 거점 주변 통제 반경
- 현상 기록/사건 좌표 주변 관측 반경

### 3. FEEDS 패널

우측 보조 HUD로 `ACTIVE FEEDS` 패널을 추가한다. 권역별로 위성, 현장 검수, 봉쇄망, 드론/해상 감시 피드를 다르게 표시한다.

### 4. 위성/드론 감시선

- 위성 링크선
- 드론 관측 경로
- 현장 감시선

기존 봉쇄선과 다른 시각 언어로 표시한다.

### 5. 줌 단계별 차등

- ZOOM 1: 해칭 오염권 중심
- ZOOM 2: 감시 반경 / 감시 링크선
- ZOOM 3: 마커 미니 카드 일부 표시
- ZOOM 4~5: 시설 셀 / 분석 카드 / 상세 관제 라벨 표시

## 제한 사항

이번 패치는 시각 업그레이드만 수행한다. 지도 조작 로직, 기록보관서, 세력관계도, F.H.C, 영상 피드, 개체 스캔, 전체 효과음 믹싱은 변경하지 않는다.
