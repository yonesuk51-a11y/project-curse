# Project Curse U.A.C Server - MapPatch5 Safe UI Reapply

## 기준
- `MapPatch4_9_FunctionRestore` 기반.
- 기본 기능을 건드리지 않고, 안전한 UI 효과만 재적용했습니다.

## 반영
- 상단 U.A.C 서버 상태 바 유지 및 브레드크럼 보강.
- 문서 상태 뱃지 유지.
- 기록 열람 로딩 최소 표시 시간과 게이지 연출 개선.
- 지역지도 권역 전환 코드 로딩 표시 시간을 너무 짧지 않게 조정.
- 지도 레이어 부팅 연출을 가볍게 유지하면서 표시 시간을 보강.
- 지도 확대 단계 표시 추가: ZOOM 1~5.
- 확대/축소/드래그/권역 전환/기록 열람/배경음 등 기존 복구 기능 유지.

## 의도적으로 제외
- F.H.C 검열 효과 본격 적용.
- 영상 손상 피드 본격 적용.
- 개체 생체 스캔 본격 적용.
- 세력 관계도 고급 모듈 재적용.
- 줌 단계별 건물/도로 상세 레이어.

위 항목들은 기본 기능 안정성 확인 후 별도 패치에서 메뉴별로 하나씩 적용하는 것이 안전합니다.
