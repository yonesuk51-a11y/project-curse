# Content Opportunity Guide / U.A.C Patch QA

컨텐츠 제안 시스템은 새 설정을 자동 확정하지 않는다. QA는 “넣으면 좋아질 후보”를 제안하고, 세계관 핵심 추가/변경은 사용자의 승인 후 반영한다.

## 제안 가능 항목

- 세계 개요: 현재 통제 상태 요약, 주요 레드존 동향, 최근 작전 로그 후보.
- 세력: 세력별 관련 기록 링크, 관련 지도 좌표, 관계 변화 기록 후보.
- 지역지도: 통신 이상권과 Ghost Channel 기록 연결, 항만 화이트존과 검문소 연결, 봉쇄선과 관측기지 연결 후보.
- 기록보관서: 관련 지도 좌표, 관련 세력, 문서 내부 목차, 손상 기록 섹션 후보.
- 오디오/연출: 제한 기록 오류음, 영상 기록 전용 VHS 효과음, 손상 기록 라인 노이즈 후보.

## 자동 확정 금지 항목

- 새 세력
- 새 존 체계
- 새 대형 사건
- 새 개체 분류 체계
- 기존 핵심 명칭 변경
