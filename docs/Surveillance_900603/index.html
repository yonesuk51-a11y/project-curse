오염 기술·감시 체계 기록

Surveillance_900603

전략 예측, 위성 감시, 기록 분석, 관제 실패 사례를 다루는 U.A.C 감시 체계 기록.

기록면을 선택해서 열람하십시오.

전략 예측

전략 예측 체계는 레드존의 확산, 타락 개체 이동, 블랙존 후보지를 계산하기 위한 U.A.C 관제 기술이다.

분석 대상

  • 타락 개체 이동 경로
  • 귀환자 이상 반응 밀도
  • 오염 무전 수신 위치
  • 혈문 반복 발생 좌표
  • 여왕형 개체 신호 후보지

주의 사항

기록 변질 구역에서는 예측 결과 역시 오염된 자료로 취급한다. 예측 결과만으로 N.H.C 투입을 결정하지 않는다.

위성 감시

위성 감시는 레드존 외곽 변화와 블랙존 후보지를 추적하는 장기 관측 체계다.

관측 항목

  • 도시 조명 소실
  • 도로 구조 불일치
  • 대형 열원 또는 생체 군집 반응
  • 지도와 실제 지형의 차이
  • 구름층 아래 반복되는 붉은 빛

궤도 관측 노드

관측 노드는 레드존 지도와 지역지도 마커의 기준 데이터를 제공하지만, 블랙존 내부에서는 좌표 신뢰도가 급격히 낮아진다.

기록 분석

기록 분석은 헬멧 카메라, 무전 로그, 차량 블랙박스, 선별소 문답 기록을 복원해 오염 징후를 찾는 절차다.

분석 항목

  • 영상 속 인물 수 불일치
  • 음성 지연과 동일 문장 반복
  • 지도 좌표 누락
  • 귀환자 진술 변조
  • 죽은 시간 구간 자동 표시

격리 기록

오염 가능성이 있는 원본 파일은 복제하지 않는다. 열람은 U.A.C 분석실 승인으로 제한한다.

관제 실패 사례

관제 실패 사례는 감시 체계가 이상현상을 늦게 감지하거나 잘못 분류한 사건을 모은 기록이다.

주요 유형

  • 오염 무전을 일반 잡음으로 오판한 사례.
  • 블랙존 후보지 예측 누락.
  • 영상 변질을 카메라 고장으로 처리한 사례.
  • 귀환자 집단 기억 동기화를 민간 소문으로 분류한 사례.

후속 조치

관제 실패는 기록 말소가 아니라 별도 격리 보관 대상이다. 반복 실패가 확인되면 해당 구역의 보안 등급을 상향한다.
