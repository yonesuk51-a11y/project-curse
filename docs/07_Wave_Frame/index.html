확장설정 07. Wave Frame / 파동기 확장
저주성 파동·압력 기반 전투 시스템.
Wave Frame은 위버멘시가 사용하는 저주성 파동·압력 기반 전투 시스템이다.
파동기는 신체 내부에 저주성 압력을 순환시키고, 이를 근육, 뼈, 신경, 무기, 방어막 파괴에 사용하는 기술이다.
사용 방식
  • 내부 순환
  • 외부 방출
  • 손바닥 방출
  • 무기 전달
  • 방어막 파괴
  • 근접 타격 강화
  • 혈무와 동기화
효과
효과 설명
신체 강화 속도, 근력, 반응속도 증가
무기 강화 혈무 절단력 증가
방어 관통 괴이 장갑, 저주 장벽 파괴
충격파 근거리 압력 방출
역장 간섭 적의 보호막 교란
오염 저항 저주성 압력 상쇄
유파
  • 정통 유파: 안정적인 내부 순환과 방어 해석을 중시한다.
  • 복합 유파: 여러 스승, 전장, 괴이와의 교전 경험을 흡수한 고속·고위험 유파.
  • 창술 계열: Blood Lance, Anchor Lance, 단창, 장창을 이용한 관통형 전투 체계.
  • 금지 유파: 여러 저주를 동시에 다루고 하나의 억제 기술로 제어하는 위험한 계열. 실패하면 사용자가 괴이화될 수 있다.
