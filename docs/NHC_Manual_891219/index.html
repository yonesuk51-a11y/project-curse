N.H.C 현장 운용 절차

레드존 진입, 생존, 블랙 태그 판정, 회수·소각 연계, 장비·시설·차량·특수탄약 운용을 묶은 현장 운용 기록면이다.

N.H.C 생존 규칙

N.H.C 생존 규칙은 훈련 교범이 아니라 반복된 작전 실패와 전사 기록에서 만들어진 현장 기준이다. 대원이 가장 먼저 배워야 하는 것은 싸우는 법이 아니라 돌아오는 법이다.
  • 혼자 진입하지 않는다.
  • 사망자 무전에 응답하지 않는다.
  • 같은 골목이 반복되면 전진하지 않는다.
  • 붉은 문이 열려 있으면 통과하지 않는다.
  • 헬멧캠 기록과 실제 위치가 다르면 즉시 후퇴한다.
  • C.P.D 보호 구역에서 무장 교전을 시작하지 않는다.
  • A.R.F 회수 루트 위에 화력을 유도하지 않는다.
  • Ash Crew 처리 구역에 불필요하게 접근하지 않는다.

블랙 태그 기준

블랙 태그 기준은 레드존 내부 사망자, 오염 사체, 회수 불가 대상, 고위험 장비를 분류하기 위한 현장 기준이다. 이 절차는 죽은 사람을 데려오기 위한 절차가 아니라 죽은 것이 다시 돌아오지 못하게 막기 위한 절차다.
- 화이트 태그: 회수 가능한 사망자나 장비.
- 그레이 태그: 제한 조건 아래 회수 가능한 대상.
- 레드 태그: 회수 위험 대상. 봉인 회수 또는 재분류 필요.
- 블랙 태그: 회수 금지 대상. 현장 봉쇄, 소각, 완전 격리 우선.
- 판정 불가 태그: 시간 오염, 위장 개체 의심, 위치 불일치 발생 시 사용하는 임시 태그.

블랙 태그 적용 조건

  • 사체가 무전으로 응답한다.
  • 사망 시간이 기록과 맞지 않는다.
  • 같은 사체가 여러 위치에서 발견된다.
  • 혈문 중심부에서 발견된다.
  • 회수 백 내부에서 움직임이 감지된다.
  • 오염 무전에 감염된 장비와 연결되어 있다.

회수·소각 연계 절차

Ash Crew는 N.H.C 산하 특수 처리반이다. 전투가 끝난 뒤 레드존 내부에 남은 오염 사체, 블랙 태그 대상, 오염된 장비, 혈문 잔류물을 봉인, 회수, 소각한다.
- 담당 임무: 오염 사체 처리, 블랙 태그 대상 소각, 오염된 장비 봉인, 혈문 잔류물 제거.
- 장비: 밀폐형 방호복, 고열 소각기, 블랙 태그 봉인 백, 오염 측정기, 오염 무전 차단기.
- 기능: 죽은 사람을 데려오기보다 죽은 것이 다시 돌아오지 못하게 막는다.

A.R.F / C.P.D 연계

A.R.F는 회수 구역에서 생존자, 장비, 샘플, 기록 매체를 회수한다.
C.P.D는 민간인 대기 구역과 선별 검사 구역에서 민간인 대피와 귀환자 분리를 담당한다.
Ash Crew는 소각 구역에서 블랙 태그 대상과 오염된 장비를 처리한다.
  • A.R.F 회수 루트와 Ash Crew 소각 루트는 겹치지 않는다.
  • C.P.D 보호 구역은 N.H.C 교전 구역과 분리한다.
  • 블랙 태그 대상은 회수 구역이 아니라 소각 또는 격리 구역으로 이동한다.

장비·시설·차량·특수탄약 운용

N.H.C 장비와 시설은 전투만을 위한 것이 아니라 오염 측정, 통신 차단, 시간 기준 유지, 민간인 분리, 회수물 봉인, 사체 처리까지 포함한다.

장비·시설 운용

  • 오염 측정기: 구역, 인간, 개체, 장비의 오염도를 측정한다.
  • 현장 동기화 단말기: 작전 기준 시간과 대원 위치를 동기화한다.
  • 오염 무전 차단기: 오염된 통신 신호를 차단한다.
  • 시간 표식탄: 시간 오염 구역에서 철수 방향을 표시한다.
  • 블랙 태그 봉인 백: 회수 금지 대상의 접촉과 확산을 막는다.
  • 휴대 소각기: 오염 잔해와 오염 장비를 현장에서 소각한다.
  • 외곽 검문소: C.P.D가 민간인 이동을 통제하는 지점.
  • 민간인 대기 구역: 피난민을 임시 대기시키는 구역.
  • 선별 검사 구역: 신원 기록과 오염 반응을 확인하는 검사 구역.
  • 작전 대기 구역: 무장, 탄약, 진입 명령이 준비되는 N.H.C 구역.
  • 회수 구역: A.R.F가 회수품과 생존자를 인계받는 구역.
  • 소각 구역: Ash Crew가 사체와 오염 장비를 처리하는 구역.
  • 지휘 컨테이너: U.A.C, N.H.C, S.I.D가 작전 정보를 공유하는 지휘소.
  • 격리실: 레드 태그 대상과 귀환자를 임시 격리하는 공간.

차량·특수탄약 운용

  • 정찰 차량: 정찰과 전방 확인에 사용된다.
  • 장갑 수송차: N.H.C 병력 수송에 사용된다.
  • 회수 차량: A.R.F 회수 작전에 사용된다.
  • 소각 장비 차량: Ash Crew의 소각 장비와 사체 처리 장비 운반에 사용된다.
  • C.P.D 대피버스: 민간인 대피와 보호소 이동에 사용된다.
  • 백염탄: 낮은 단계의 오염 개체와 잔류물에 사용하는 제압탄.
  • 소각 봉쇄탄: 소각과 확산 차단을 동시에 목적으로 하는 탄약.
  • 혈액 억제탄: 혈액성 변이 개체의 재생 반응을 억제하는 탄약.
  • 고정 표식탄: 시간 오염 지역에서 위치 기준을 남기는 표식탄.
  • 혈문 억제 장약: 혈문 주변 잔류 오염을 줄이는 폭발식 장비.

운용 제한

  • 레드존 내부 차량은 단독 운행하지 않는다.
  • 죽은 시간 의심 구역에서는 자동항법을 사용하지 않는다.
  • 오염 무전 발생 시 차량 무전망을 분리한다.
  • 블랙 태그 대상이 실린 차량은 일반 기지로 복귀하지 않는다.
  • 오염 차량은 수리 후 재투입하지 않는다.

N.H.C 봉쇄·사후 대응 절차

봉쇄선 붕괴 상황, 실패 작전 기록 회수, 국외 레드존 공동 대응 기준을 묶은 사후 대응 기록면이다.

봉쇄선 붕괴 코드

봉쇄선 붕괴 코드는 봉쇄선이 무너질 때 사용하는 긴급 분류 코드다. 단순 침입 경보가 아니라 타락 개체, 민간인, 시간 오염, 혈문, 블랙 태그 대상의 영향이 봉쇄선 밖으로 번질 가능성을 표시한다.
붕괴 코드 0: 오경보.
붕괴 코드 1: 미세 누출. 오염 신호나 혈흔이 외곽에서 확인됨.
붕괴 코드 2: 개체 접촉. 타락 개체 또는 오염 개체가 봉쇄선과 접촉함.
붕괴 코드 3: 민간인 돌파. 귀환자 또는 감염 의심자가 통제선을 넘음.
붕괴 코드 4: 혈문 발생. 혈문 또는 유사 문 현상이 발생함.
붕괴 코드 5: 죽은 시간 변동. 봉쇄선 주변 시간이 뒤틀림.
붕괴 코드 6: 블랙 태그 확산. 회수 금지 대상의 영향이 외곽으로 번짐.
붕괴 코드 7: 블랙존화 위험. 구역 자체가 블랙존 전환 위험에 들어감.

작전 실패 후 기록 회수 절차

작전 실패 후 기록 회수 절차는 실패한 작전, 대량 전사, 블랙존 전환, 민간인 손실, 비인가 연구 개입과 관련된 기록을 U.A.C가 회수, 봉인, 제한 열람 처리하는 절차다.
  • 헬멧캠 영상, 무전 로그, 작전 지도, 사망자 명단을 회수한다.
  • 오염 무전에 감염된 파일은 재생 금지 상태로 봉인한다.
  • F.H.C 샘플 기록과 Amarion 회수 영상은 별도 보안 저장소로 이동한다.
  • 블랙 태그 대상과 연결된 기록 매체는 Ash Crew 처리 대상이 될 수 있다.
  • 말소된 기록은 죽은 시간 구역에서 다시 반복 재생될 수 있다.

국외 레드존 대응 기준

국외 레드존 대응 기준은 레드존이 도시, 국가, 국경을 넘어 확산될 때 적용되는 제한 대응 체계다. 타락 개체 이동뿐 아니라 오염 장비, 귀환자, 기록 매체, 무전 신호, 연구 샘플의 이동까지 통제한다.
- 지역 레드 사건: 한 도시나 제한된 지역에서 발생한 레드존 사건.
- 국가 레드 위기: 국가 내부 여러 지역으로 오염이 확산되는 상황.
- 국경 오염 확산: 국경을 넘어 오염자, 장비, 신호, 타락 개체 흔적이 이동한 상황.
- 국제 블랙존 위험: 블랙존 전환 가능성이 국제 단위로 확산되는 상황.
