N.H.C 현장 작전·장비·봉쇄 규정 문서

NHC_Manual_891219

레드존에서 회수된 장비 기록과 미복귀 대원의 마지막 보고를 바탕으로 작성된 N.H.C 내부 규정. 진입, 장비 운용, 봉쇄 실패 이후의 기록 회수 절차가 현장 대원용으로 정리되어 있다.

기록면을 선택해서 열람하십시오.

현장 운용

레드존에 투입되는 N.H.C 대원은 구조보다 봉쇄, 회수보다 생존, 확인보다 기록 보존을 우선한다. 현장 판단은 생존 규칙, 블랙 태그, 회수·소각 연계 절차에 따라 순차적으로 진행된다.

N.H.C 생존 규칙

  • 단독 진입 금지. 2인 이하 진입조는 레드존 내부 작전으로 인정하지 않는다.
  • 죽은 시간 발생 시 이동을 중지하고 마지막 정상 좌표를 반복 송신한다.
  • 혈문, 반복 복도, 오염 무전 발생 지점은 추적하지 않는다.
  • 귀환자 발견 시 구조보다 C.P.D 인계를 우선한다.
  • 위장 개체 의심 시 암구호와 생체 신호를 동시에 확인한다.

블랙 태그 기준

  • 회수 불가 사체, 오염 장비, 반복 음성 송신 장비, 둥지 내부 잔류물에 부착한다.
  • 여왕형 개체 신호와 혈문 잔류가 동시에 확인된 대상은 블랙 태그 후보로 둔다.
  • 블랙 태그 부착 후에는 A.R.F 단독 회수 금지, Ash Crew 호출을 우선한다.

회수·소각 연계 절차

  • A.R.F는 샘플·기록·생존 가능 대상만 회수한다.
  • Ash Crew는 오염 사체, 파손 장비, 회수 금지 물품을 봉인·소각한다.
  • 귀환자와 오염 사체가 혼재된 현장에서는 C.P.D 선별소와 임시 격리실을 먼저 확보한다.

장비·자산 운용

장비·자산 운용 기록면은 N.H.C 현장에서 실제 지급, 제한 지급, 회수 금지로 나뉘는 자산을 관리한다. 장비의 기원이나 제작 원리는 별도 특수 병기·시설 운용 기록으로 이관한다.

표준 화기

  • 제식 소총, 산탄총, 기관총, 저격 장비는 외곽 방어와 통로 제압에 사용한다.
  • 타락 개체 군집 교전 시 탄약 소모량을 기준으로 철수 시간을 계산한다.

특수탄약

  • 오염 억제탄: 체액 반응과 재생 속도를 낮춘다.
  • 재생 억제탄: 상위 개체 후보의 회복을 지연시킨다.
  • 소각탄: 잔류 조직과 오염 장비 처리에 사용한다.
  • 표식탄: 회수 불가 대상과 둥지 입구를 원거리 표시한다.
  • 봉쇄 장약: 혈문 주변 구조물과 오염벽을 차단한다.

근접·방호 장비

  • 혈무는 일부 분대장, 특수 대응반, 위버멘시 협동 작전 인원에게 제한 지급된다.
  • W-Coat와 W-Fiber 장비는 고오염 구역 단기 진입에만 사용한다.
  • Null 코팅 장비는 혈문 주변 부착 오염과 의식성 표식을 지연시킨다.

차량·시설 운용

  • C.P.D 대피버스는 민간인 이송과 귀환자 분리 수송을 담당한다.
  • 회수 차량과 이동 격리차량은 A.R.F 승인 좌표 밖으로 진입하지 않는다.
  • 외곽 검문소, C.P.D 선별소, A.R.F 회수 구역, Ash Crew 소각 구역은 지도 마커와 함께 관리한다.

봉쇄·사후 대응

봉쇄·사후 대응은 작전이 정상 절차를 벗어났을 때 발령된다. 이 기록면은 봉쇄선 붕괴 코드, 실패 작전 기록 회수, 국외 레드존 공동 대응 기준을 다룬다.

봉쇄선 붕괴 코드

  • 1단계: 민간인 대피 지연, 외곽 검문소 포화.
  • 2단계: 타락 개체 외부 이동, 방어선 일부 파손.
  • 3단계: 오염 무전 반복, 혈문 발생, 장비 기록 지연.
  • 4단계: 여왕형 개체 신호 의심, 둥지화 진행, 방어선 전면 철수.
  • 5단계: 블랙존화 후보 선포, 회수 중단, 기록 보존 우선.

작전 실패 후 기록 회수 절차

  • 미복귀 대원의 헬멧 카메라, 무전 로그, 위치 기록을 우선 회수한다.
  • 오염 무전 원본은 현장 재생을 금지하고 U.A.C 분석실에 이관한다.
  • 지도 좌표가 변질된 기록은 복사본 제작을 금지한다.
  • 회수 불가능한 기록 장비는 블랙 태그 부착 후 소각한다.

국외 레드존 대응 기준

  • 국경 인접 레드존은 U.A.C 국제 감시 체계와 공동 판정한다.
  • N.H.C 파견은 현지 정부 작전권과 충돌하지 않는 범위에서 제한 승인한다.
  • 블랙존 후보지와 여왕형 개체 신호 기록은 공유 범위를 제한한다.
