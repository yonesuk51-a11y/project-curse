확장설정 06. Wivermensi 확장
저주성 힘, 신체 강화, 장비, 혈무, 파동기의 결합체.
Wivermensi / 위버멘시는 저주성 힘, 신체 강화, 전술 장비, 혈무, 파동기 시스템이 결합된 강화 인간 요원이다.
위버멘시는 단순한 슈퍼솔저가 아니라, 우시노다 계열 저주력을 무기화한 인간형 특수 병기에 가깝다.
구성 요소
  • W-Fiber
  • W-Coat
  • Wave Frame
  • Null Coating
  • Blood Weapon / 혈무
  • 신경 조건화
  • 저주 내성 조정
  • 육체 재구성
위버멘시 계급
계급 특징
Low Class 제한적 강화, 높은 사망률
Standard Class 일반 작전 투입 가능
High Class 레드존 심부 진입 가능
Master Class Superior-Type과 교전 가능
Nullklasse 블랙존 및 특수 이상현상 대응 가능
문제점
  • 인간성 약화
  • 감각 왜곡
  • 혈무 동기화 실패
  • 파동기 과부하
  • 신체 재구성 부작용
  • 저주성 힘에 의한 내부 오염
  • 명령 불복종 혹은 폭주 가능성
