확장설정 05. 귀환자 확장
레드존이나 블랙존에서 돌아온 인간의 판정 체계.
귀환자는 레드존이나 블랙존에서 실종된 뒤 다시 돌아온 민간인 혹은 병력이다.
  • 기억 일부 손실
  • 신체 오염 반응 없음으로 보일 수 있음
  • 특정 단어, 장소, 의식 문장에 반응
  • 꿈, 환청, 검은 액체, 반복 행동 보고
  • 민간인으로 복귀 가능한지 논란
C.P.D와 S.I.D가 공동으로 판정한다.
