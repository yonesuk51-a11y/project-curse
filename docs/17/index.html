확장설정 17. 최종 정리
Project Curse 확장설정의 핵심 문장.
Project Curse는 괴이와 인간이 싸우는 세계가 아니라, 괴이를 연구하고, 이용하고, 은폐하고, 무기화하다가 결국 도시와 인간성이 함께 무너지는 세계관이다.
  • U.A.C는 통제한다.
  • N.H.C는 죽으면서 막는다.
  • S.I.D는 진실을 추적한다.
  • F.H.C는 실험한다.
  • 기업은 이용한다.
  • 종교 집단은 숭배한다.
  • 괴이는 번식한다.
  • 위버멘시는 인간과 병기 사이에서 버틴다.
  • Young Soldier는 전쟁의 소모품이 된다.
  • 귀환자는 인간인지 아닌지 판정받는다.
  • Redwolf는 체제의 실패를 증명한다.
  • 신디케이트는 재난을 시장으로 만든다.
  • 중앙 재난 의회는 도시의 생사를 결정한다.
  • 군산복합 기업은 전쟁이 끝나지 않기를 바란다.
  • 블랙존 생존자는 인간 사회의 정의를 흔든다.
  • 레드존은 도시를 먹고, 블랙존은 현실을 삼킨다.
