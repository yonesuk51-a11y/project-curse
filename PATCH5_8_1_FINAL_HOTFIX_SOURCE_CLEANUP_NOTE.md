# PATCH 5.8.1 — FinalHotfix SourceCleanup

기준 버전: `MapPatch5_8_FinalUIPerformanceQA_FactionFocusPolish`

## 목적

5.8 적용 후 마감 전 단계에서 남은 잔버그와 배포 구조를 정리하는 최종 핫픽스다. 새 기능을 크게 추가하지 않고, 세력관계도 포커스 가독성과 GitHub 업로드 기준 소스 정리를 우선한다.

## 포함 사항

### 1. 세력관계도 포커스 명도 재조정

- 선택 노드는 최상위 밝기로 유지.
- 직접 연결 노드는 선명하게 유지.
- 2차 연결 노드는 중간보다 낮은 명도로 후퇴.
- 무관 노드는 5.8보다 더 투명하게 후퇴.
- 무관 노드의 마크, 이름, 상태 태그도 함께 낮은 명도로 처리.

### 2. 직접 연결선만 표시

포커스 상태에서는 선택한 노드에 직접 붙은 관계선만 표시한다.

- 선택 노드 ↔ 직접 연결 노드: 표시
- 직접 연결 노드 ↔ 다른 직접 연결 노드: 숨김
- 2차 연결 노드 ↔ 외곽 노드: 숨김
- 무관 노드끼리의 선: 숨김

이 규칙으로 U.A.C를 클릭하면 U.A.C에서 뻗는 관계만 남고, 다른 세력끼리의 선은 관계도 배경에서 사라진다.

### 3. 관계표 가독성 보정

- 선택 세력과 직접 연결된 관계표 행은 강조.
- 선택 세력 주변 관계는 낮은 명도.
- 무관 관계 행은 더 투명하게 후퇴.

### 4. 배포 기준 정리

- 최신 배포 기준을 README에 반영.
- 이전 패치 ZIP을 소스에 포함하지 않는 방향 유지.
- 대용량 원본 참고 파일은 현재 UI 소스에 포함하지 않는 방향 유지.
