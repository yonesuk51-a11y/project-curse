- 종교 첫 화면을 F.H.C 반부패부서 기밀 경고문으로 교체하고 비교·현장 대응 요약 화면을 제거.
- 타락교 현장 프레임과 종교 전체 길이 배경음을 새 원본으로 교체.
- 괴이의 중간 분류 해설과 마지막 원칙·대응 지침을 제거하고 15개 회수 기록을 연속 재생하도록 정리.
- `그림자의 희생자 / VIDEO-1092C1F2`를 그림자 기록 다음에 추가하고 원문 두 문장을 번역해 수록.
- 종교 영상과 중복되는 `가면을 쓴 존재 / IMAGE-0321`를 괴이 분류 기록에서 제외.

## ENEX Source Restoration — Feral Classification

- `괴이`를 8개 장문 항목과 확인 개체 15종으로 복원.
- Ferals / Pure·Unpure, Superiors / Celestials·Odious, Unusuals / Artificial·Hybrid의 원문 체계를 유지.
- Celestials·Odious가 우시노다교 계급이 아닌 과거 현장 하위분류임을 명시.
- 봉인을 오토마톤 작동·증폭에 사용되는 마법 조작 도구로 원문에 맞춰 복원.
- 귀환자·방랑자를 괴이로 자동 판정하지 않는 현행 판정 원칙을 반영.
- ENEX 원본 이미지 18장을 분류도·관측 프레임·개체별 항목에 연결.
- 개체 분류 용어에 계열별 색상 표기를 추가.
- 장문 문서를 종교 기록과 같은 전체화면 시퀀스 19면으로 재구성.
- 종교 기록의 도입 영상·VHS 계열 전환·페이지 효과음을 동일하게 재사용.
- 개체별 이미지와 본문을 한 기록면씩 보여 주며 이전·다음·일시정지·처음부터 탐색을 지원.
- 전용 배경음 `you're inside the last memories of a dying person` 연결 경로를 추가.
- 시퀀스 시작 즉시 메뉴 배경음을 정지해 기록 배경음과 겹치지 않도록 수정.
- 종교·타락 개체 보고서에 동일한 라디오 정적음 레이어를 추가.
- `괴이`의 열람 직후 도입 영상을 종교와 같은 원본 손상 신호 영상으로 통일.
- `괴이`를 기록보관소 내부 시퀀스로 전환해 브라우저 자동재생 차단으로 도입 영상이 생략되던 문제 수정.
- 구형 공개 기록 잠금 래퍼가 `괴이`의 내부 시퀀스 호출을 거부하던 충돌을 제거하고, 색인 버튼에서 영상 런타임을 직접 실행하도록 수정.
- 작성 경위와 Ferals·Superiors·Artificial·Hybrid 장 구분을 추가하고, 개체별 내용을 분류·위험 구역·개요·식별 단서·현장 대응 순으로 통일.
- 반복적인 요약 표현을 줄이고 관찰 사실·미확인 사항·현장 지시가 구분되도록 `괴이` 본문 문체 정리.
- 원본 `FERALS` 소개 프레임을 미확인 개체가 아닌 괴이 분류 소개면으로 재배치.
- Ferals의 한국어 표기를 `야성체`에서 `괴이`로 통일.
- 원본 `FERALS` 소개문을 번역해 개체 수·포식 방식·불명확한 기원·주술적 연관성을 두 문단으로 복원.
- `개 / IMAGE-007CF` 항목에서 원문에 없던 위험도·출현 구역·현장 대응을 제거하고, 야생 습성·생체 흡수·불멸성에 관한 원문 번역만 남김.

- 독립 설정글에서 기록보관소로 복귀할 때 메인 부팅 연출을 건너뛰도록 수정.

## ENEX Source Restoration — N.H.C Field Manual

- `N.H.C 현장 작전·장비·봉쇄 규정 문서`를 10개 장문 항목으로 복원.

- Ash Crew·A.R.F·C.P.D의 처리선·회수선·민간선과 임시기지 구획을 정리.
- 원본 개별 장비명은 기존 여섯 장비군 아래의 현장 명칭으로 통합.
- ENEX 원본 이미지 6장을 교범·Black Tag·세 조직·장비 항목에 연결.

## ENEX Source Restoration — Zone / Redzone

- `구역 위험도 분류 문서`를 ENEX 원문 기준의 장문 통합 개정본으로 복원.
- 화이트존을 안전 구역이 아닌 정보 부족·판정 불가 구역으로 바로잡음.
- U.A.C의 정식 명칭과 독립기관 성격, 2001년 N.H.C·S.I.D 독립 개편, 2005년 Ash Crew 체계를 반영.
- 다섯 구역의 기관 책임, 출입 권한, 격상·하향 조건과 Level 0–7 보안 등급을 복원.
- `레드존 이상현상 및 오염 기준 문서`에 CI-H/Z/E/O/S, CI-0–7, Dead Hour, C.A.P-17, Blood Gate, Ghost Channel, 장기 방치 단계를 복원.
- ENEX 원본 이미지 7장을 항목별 본문에 연결하고 패키지용 WebP로 최적화.
- 장문 표, 하위 항목, 회수 이미지와 용어별 색상 표기를 지원하도록 공용 기록 렌더러를 확장.

## Archive Consolidation

- `archive-registry.js`에 대표 기록철 4개와 세부 문서 소속을 정의.
- 기록보관소 목록을 대표 기록철 4개 카드로 재구성.
- 기존 현상·세력·장비 문서를 해당 기록철의 접히는 기록면으로 통합.
- 사건·회수·개체·규정의 제한 기록은 대표 기록철 안에서 봉인 색인으로 정리.
- 분산 카드가 구형 렌더러에 의해 다시 추가되면 대표 목록으로 되돌리는 보정 추가.
