# PATCH 5.8.1R-52 — TacticalDisplayDecongestionPass

## 목적

R51 이후 화면 검토에서 확인된 전술 오버레이 과밀 문제를 줄이고, 일반 작전 화면과 검수 화면을 분리한다.

## 변경

- 일반 화면의 BARRIER / OP-GROUP 계열 긴 라벨을 숨김 또는 줌5/선택/검수 조건으로 제한.
- 관계권 프레임과 연결선의 불투명도와 두께를 낮춤.
- 미니 정보 카드를 줌5/검수 중심으로 제한하고 크기를 축소.
- 시설·사건·통신·봉쇄 거점 아이콘과 점 표시 크기를 축소.
- SELF REVIEW, FINAL CHECK, VISIBLE NODES 목록을 검수 모드 전용으로 분리.
- RECORD INDEX 일반 화면 기본 표시량을 축소하여 패널 스크롤 부담을 완화.
- R50/R51의 휠 줌, 드래그, 선택 위치 유지, 관계권 중심 기능은 유지.

## 유지

- 세계지도 권역 선택 전용 구조 유지.
- 줌 1~5 단계형 표시 유지.
- 전체 지도 요소 165개 유지.
- 정밀 전용 데이터 0개 유지.
- 새 존 체계 추가 없음.
