# PATCH 5.8.1R-42 — MapPerformanceAndIndexPolish

R41에서 확장된 149개 지도 요소를 기준으로 RECORD INDEX, 겹친 요소 후보 메뉴, SELF REVIEW/최종 점검 리포트의 가독성과 처리 부담을 정리한 패치입니다.

## 핵심 변경

- 지도 모듈 버전을 `5.8.1R-42 MapPerformanceAndIndexPolish`로 갱신했습니다.
- RECORD INDEX에 간단한 캐시 계층을 추가해 동일 조건의 색인 계산 반복을 줄였습니다.
- 색인 표시 개수를 기본/필터/검색 상황별로 제한해 목록이 과도하게 길어지지 않도록 정리했습니다.
- 색인 결과에 현재 권역 우선 표시, 선택 항목 강조, 유형 라벨 표시를 추가했습니다.
- 겹친 요소 후보 메뉴를 최대 표시 개수 기준으로 압축하고, 후보 유형/관계권/검수 ID 표시를 정리했습니다.
- SELF REVIEW 및 최종 점검 리포트에 성능·색인 요약을 추가했습니다.
- RECORD INDEX/후보 메뉴 관련 CSS를 보강해 149개 데이터 기준에서도 패널이 과하게 늘어나지 않도록 조정했습니다.

## 유지 사항

- R41 콘텐츠 확장 데이터 유지
- 새 존 체계 추가 없음
- 세계지도 권역 선택 전용 구조 유지
- 지도↔문서 연결, LINK AUDIT, RECORD INDEX, SELF REVIEW 유지
- 일반 화면 내부 ID 비노출 / 검수 모드 내부 ID 표시 유지

## 검수 기준

- JS 문법 체크 통과
- 데이터 모듈 문법 체크 통과
- RECORD INDEX 검색/필터/관계권 이동 정상
- 겹친 요소 후보 선택 정상
- 최종 점검 리포트 정상
