# Project Curse U.A.C Server Archive

이번 수정본은 추가 확장설정과 추가 이미지 삽입분을 제거하고, 지역지도를 실제 세계지도 기반 대륙권 기록면으로 다시 구성한 버전입니다.

## 적용 내용
- 추가 확장설정 문서 5개 제거 상태 유지
- 최근 업로드 이미지 자동 삽입분 제거 상태 유지
- 지역지도: 실제 세계지도 기반으로 재작성
- 지역지도 기록면: 세계 / 동아시아 / 유럽 / 북미 / 남방권 / 중동·아프리카
- 각 기록면 내부 필터: 전체 / 오염 구역 / 작전 시설 / 현상 기록 / 사건 좌표 / 봉쇄선
- 바다 영역은 색 면적 대신 점선 감시권·사건 좌표 방식으로 표시
