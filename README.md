# Project Curse U.A.C Server - HTML Native Final

GitHub Pages용 Project Curse U.A.C 정보 서버 최종 수정본입니다.

## 핵심 수정
- 시작 화면, 세계지도, 세력 관계도, Archive Floor를 통짜 이미지로 쓰지 않습니다.
- GLOBAL MAP은 HTML/SVG 지도 레이어로 구성됩니다.
- RELATION MATRIX는 HTML/SVG 노드 관계도로 구성됩니다.
- Archive Records는 CSS/HTML 문서 카드로 구성됩니다.
- 이미지는 RECOVERED FRAMES와 원본 회수 프레임 자료에만 사용됩니다.
- 지속 배경음, hover 사운드, 메뉴 이동 사운드는 제거되어 있습니다.
- 비밀번호는 `Curse` 입니다.

## 업로드 방법
압축을 푼 뒤 이 폴더 안의 `index.html`, `archive`, `docs`, `assets`, `README.md`, `.nojekyll`을 GitHub 저장소 루트에 넣으세요.
