# Project Curse U.A.C 기록 단말기

GitHub Pages 업로드용 정적 사이트 패키지입니다.

## 적용 내용
- 비밀번호 화면 제거
- 로딩 후 세계 개요만 표시
- 왼쪽 메뉴 방식으로 화면 전환
- 홈에서 모든 섹션이 한꺼번에 보이지 않도록 수정
- 인물 프로필 기록과 연표 기록 문서 제외
- 연표 내용은 세계 역사 화면에 통합
- 확장 설정 문서 제외
- 주요 세력 기록서는 세력 정보 화면에서 마크 클릭 방식으로 열람
- 기록 보관소 문서는 원본 설정글 중심으로 보존
- 지정 문서는 제목 단위 기록면으로 재분할
- 이전/다음 기록면 버튼 제거
- 기록면 선택 버튼만 유지
- 첨부 이미지 빠른 보기 제거
- 특수문자와 메타 문구 정리
- 효과음 기본 켜짐

## 업로드 방법
압축을 풀고 안의 내용물만 GitHub 저장소 루트에 넣으세요.

