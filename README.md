# Project Curse U.A.C Legacy 2003 Terminal

GitHub Pages 업로드용 패키지입니다.

## 반영 내용
- 비밀번호 화면 제거
- 로딩 후 바로 `세계 역사 / 세계 개요` 표시
- 상단 메뉴가 아니라 왼쪽 메뉴형 구형 PC 단말기 UI
- 2000년대 초반 CRT/구형 기록 단말기 느낌으로 색상과 레이아웃 수정
- 한국어 중심 메뉴와 설명
- 세력 정보 안에 UAC, NHC, SID, FHC, Amarion, Syndicate, Ushinoda Cult, Haimun 마크 적용
- 원본 설정글과 확장 설정글을 개별 문서로 유지
- 지역지도에 세계지도 윤곽, Red Zone, 본부/기지/회수 허브/검문소 표기 유지
- 오디오 추가: 낮은 PC/CRT 배경음, 메뉴 이동음, 기록 열람음, 페이지 넘김음

## 업로드 방법
압축을 풀고 폴더 안의 `index.html`, `archive/`, `docs/`, `assets/`, `.nojekyll`, `README.md`를 저장소 루트에 넣으면 됩니다.

브라우저 자동재생 정책 때문에 오디오는 사용자가 AUDIO 버튼을 누른 뒤 재생됩니다.


## 사운드 수정
- 지속 배경음은 제거했습니다.
- 효과음 버튼을 켜면 부팅음, 메뉴 클릭음, 기록 열람음, 페이지 넘김음만 짧게 재생됩니다.
- 브라우저 자동재생 제한 때문에 사용자가 버튼을 한 번 누른 뒤부터 재생됩니다.


## 최종 수정 반영
- 비밀번호 화면 없음.
- 로딩 후 세계 개요만 표시.
- 좌측 메뉴: 세계 역사 / 세력 정보 / 세력 관계 / 지역지도 / 기록 보관소.
- 인물 프로필 기록과 Project Curse 연표 기록은 개별 문서에서 제외. 연표 핵심은 세계 역사에 통합.
- 주요 장문 설정글은 페이지식 열람으로 분할.
- 첨부 이미지 빠른보기 제거.
- 주요 세력 기록서는 세력 마크 클릭식 설명 패널 추가.
- 효과음은 메뉴 이동, 기록 열람, 페이지 넘김, 부팅음만 사용.
