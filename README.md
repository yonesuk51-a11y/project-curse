# Project Curse U.A.C Legacy Record Terminal

GitHub Pages 업로드용 정적 사이트 패키지입니다.

## 구성
- `index.html` : 로딩 후 세계 개요가 먼저 표시되는 메인 화면
- `archive/` : 기록 보관소
- `docs/` : 원본 설정글 기반 개별 기록 문서
- `assets/` : 스타일, 스크립트, 이미지, 효과음

## 반영 사항
- 비밀번호 화면 제거
- 왼쪽 메뉴 기반 화면 전환
- 세계 개요 단일 첫 화면
- 세력 정보 마크 클릭식 설명 패널
- Ash Crew, A.R.F, C.P.D 세력 마크 추가
- 기록 보관소 기록면 버튼 복구
- 이전/다음 페이지 버튼 제거
- 지역 지도 화면 정리
- 메뉴, 기록 열람, 기록면 선택 효과음 적용
- 배경음 제거

## 업로드
압축을 푼 뒤 폴더 안의 내용물을 GitHub 저장소 루트에 넣으면 됩니다.
