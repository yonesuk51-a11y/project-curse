## Scope-locked update

이번 패키지는 기존 ScopeLocked 기준 파일에서 아래 항목만 제한적으로 수정했다.

- 기록 보관소 구조는 유지하고, 영어 용어만 한국어 표기로 통일했다.
- 세력 관계도 제목의 제작 설명 문구를 제거했다.
- 세력 관계도는 U.A.C 중심 트리형으로 재배치해 선 겹침을 줄였다.
- 지역 지도는 사각형 영역 대신 원형 존 영향권으로 표시했다.
- 지도 내부 텍스트는 제거하고, 지도 아래 버튼형 설명 패널로 이동했다.
- 오디오 파일과 효과음 파일은 변경하지 않았다.
