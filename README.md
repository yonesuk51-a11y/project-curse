# U.A.C Information Server - Project Curse Archive

GitHub Pages용 정적 웹사이트 패키지입니다.

- Project Curse 로고 이미지는 사용하지 않고 U.A.C 보안 정보 서버 UI로 구성했습니다.
- 비밀번호는 `Curse`입니다.
- 설정글은 긴 원문 대신 세력, 역할, 관계, 지도, 기록물 카테고리 중심으로 압축했습니다.
- 기존 Audio Log MP3는 제거했습니다.
- 페이지 전환/문서 열람 효과음은 아날로그 컴퓨터/슬라이드 프로젝터 분위기로 합성했습니다.
- 이미지 리소스는 WebP로 압축했습니다.

GitHub에 올릴 때는 이 폴더 안의 `index.html`, `assets/`, `.nojekyll`, `README.md`를 저장소 루트에 업로드하세요.
