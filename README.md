# Project Curse - U.A.C Secure Archive Server (GitHub Ready Final)

이 패키지는 GitHub Pages에 바로 올릴 수 있는 최종 정리본입니다.

## 포함 사항
- U.A.C 인증 게이트 / 비밀번호 `Curse`
- START HERE / 추천 열람 순서
- Zone System / Global Redzone Map
- Faction Database / Relation Matrix
- Archive Records / Search & Tags / Related Files
- Entity Index / Weapon & Research / Field Manuals / Black Files
- Recovered Frames 갤러리
- Glossary / Server Log / Case Chain
- 지속 배경음 제거, hover 사운드 제거
- 이벤트 효과음만 선택 재생
- 설정글 내부 글자색, 글자 크기, 줄간격, 이미지 크기 통일
- 회수 이미지 캡션, Visual Record Cache, Black File 경고 게이트

## 업로드 방법
1. ZIP 압축을 풉니다.
2. `Project_Curse_UAC_Server_GitHubReady_Final` 폴더 안의 내용물을 GitHub 저장소 루트에 복사합니다.
3. GitHub Desktop에서 Commit 후 Push 합니다.
4. GitHub Pages는 `main / root`로 설정합니다.
