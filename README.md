## MapPatch5_8_1R-48 DetailMapPanAndWheelZoomControlPass

R48에서는 상세 권역지도에 한정해 조작 편의성을 보강했다.

- 세계지도는 권역 선택 전용으로 유지한다.
- 세계지도에는 줌/드래그/FIT/WIDE를 추가하지 않는다.
- 상세지도에서는 마우스 휠로 `줌 1 ~ 줌 5` 단계를 전환할 수 있다.
- `줌 1`은 전체 작전도 고정 상태다.
- `줌 2 ~ 줌 5`에서는 빈 지도 영역을 드래그해 화면을 이동할 수 있다.
- `줌/위치 초기화` 버튼으로 줌 1 및 기본 위치로 복귀한다.
- 선택 중심/관계권 중심 보기, RECORD INDEX, LINK AUDIT, SELF REVIEW 기능은 유지한다.
