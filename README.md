# Project Curse U.A.C Closed Server

Current build: `5.15.2cs Terminal Hub`

좌측 사이드 메뉴와 메뉴 버튼을 제거하고 `단말 상태`를 화면 이동의 중심으로
다시 작성했다. `app-shell.js` 하나만 허브 버튼과 화면 복귀를 관리한다.

정적 PC·모바일 기록 단말 패키지다. 루트의 `index.html`을 열거나 폴더 전체를 GitHub Pages에 배포한다.

## 현재 화면

- 단말 상태
- 세계 사건 연표
- 세력 분석실
- 기록보관소

폐기된 지역 상황도와 독립 관계도 화면은 루트 구조에서 제거했다. 예전 주소 `#region-map`은 세계 사건 연표로, `#faction-relation`은 세력 분석실로 연결된다.

기록보관소는 `종교`, `불멸을 향해`, `괴이`, `사쿠마의 테이프`를 영상 기록으로,
나머지 공개 기록을 문서 파일로 분리한다. 허브에서 화면을 선택할 때 별도의 메뉴
효과음은 재생하지 않는다.

이번 정리에서 참조되지 않는 원본 음원 2개와 이미지 5개, 폐기 화면 전용 안내 문서를 제거했다. 보호 기록인 `종교`와 `불멸을 향해`의 인라인·독립 페이지는 해시로 보존한다.

영상 기록은 공통 재생 엔진 하나와 기록별 설정 모듈 네 개로 분리했다. 종교·불멸을 향해·괴이·사쿠마의 테이프는 각자 영상, 배경음, 전환음과 페이지 공급 정보를 소유하며 `record-cinematic-runtime.js`는 등록된 설정을 조회해 재생만 담당한다.

PC와 모바일은 같은 화면 구조를 사용한다. 첫 부팅은 이전 주소의 해시와 관계없이
`단말 상태`로 진입하며, 허브의 세 버튼으로 연표·세력 분석실·기록보관소에 들어간다.
각 화면 상단의 `단말 상태로 돌아가기`로 허브에 복귀한다. 문서 페이지에서 돌아오는
`?return=archive`만 기록보관소를 복원한다. 사이드 메뉴가 사라진 공간은 연표·세력 카드·
기록 색인이 더 넓게 사용하고, 긴 본문은 읽기 폭을 유지한다.

## 공개 배포 전 확인

포함된 음원과 영상은 파일명만으로 공개 재배포 허가를 판단할 수 없다. GitHub Pages 공개 전 원본 출처, 라이선스, 출처 표기와 재배포 허용 범위를 확인한다.
