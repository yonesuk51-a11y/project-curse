# PATCH 5.8 — FinalUIPerformanceQA + Faction Focus Polish

## 기준 버전
- `MapPatch5_7_1_RestrictedRecordFXAudit_FactionFocusFix`

## 목적
원래 다음 순서였던 최종 UI/성능/회귀 검수 단계에 세력관계도 포커스 가독성 보정을 함께 포함했다.

## 포함 내용

### 1. 세력관계도 포커스 가독성 보정
- 선택 노드: 가장 선명하게 유지
- 직접 연결 노드: 선명하게 유지
- 2차 연결 노드: 중간 명도
- 연결 안 된 노드: 더 투명하게 후퇴
- 완전 숨김은 금지하되, 배경 요소처럼 보이게 조정

### 2. 연결 안 된 관계선 제거
- 선택 노드와 관계 없는 선은 표시하지 않음
- 직접 연결선만 강하게 표시
- 직접 연결 노드 사이의 보조 관계선은 낮은 명도 유지
- 관계 없는 노드끼리 이어지는 선은 숨김

### 3. 최종 회귀 검수 범위
- 메뉴 진입 연출
- 기록보관서 연출 분리
- 지역지도 줌 단계식 조작
- 지도 마커/기록 연결
- F.H.C/영상/개체 특수 연출 범위 제한
- 세력관계도 노드 선택/관계표 연동

## 검수 포인트
- U.A.C 클릭 시 직접 연결 세력이 선명하게 드러나는지
- 직접 연결이 아닌 세력은 더 투명해지는지
- 연결 안 된 세력끼리의 선이 보이지 않는지
- 관계 분석 패널이 정상 갱신되는지
- 다른 메뉴/지도/기록 기능이 회귀하지 않았는지
