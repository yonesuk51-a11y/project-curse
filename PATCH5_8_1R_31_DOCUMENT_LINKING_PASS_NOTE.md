# PATCH 5.8.1R-31 — DocumentLinkingPass

## 목적

R24~R30에서 안정화한 새 지역지도 모듈을 기존 문서/기록 구조와 연결한다. 이번 패치는 새 설정이나 새 존 체계를 추가하지 않고, 지도 요소를 선택했을 때 관련 기록을 바로 열람할 수 있도록 연결 UI를 보강한다.

## 변경 사항

### 1. 선택 요소 패널에 관련 기록 섹션 추가

지도 요소를 클릭하면 우측 선택 패널에 `관련 기록` 영역이 표시된다.

- 존
- 작전 시설
- 현상 기록
- 사건 좌표
- 봉쇄 거점
- 봉쇄선
- 통신권

각 요소의 유형, 존 등급, 관계권, 기록 문자열을 기준으로 관련 기록을 자동 산출한다.

### 2. 기존 기록 보관서와 연결

내부 기록 보관서에 존재하는 기록은 같은 화면 안에서 기록 보관서로 전환한 뒤 해당 기록을 연다.

주요 연결 대상:

- `Redzone_881120` — 레드존 이상현상 및 오염 기준 문서
- `NHC_Manual_891219` — N.H.C 현장 작전·장비·봉쇄 규정 문서
- `FCR_Archive_890402` — 타락 개체 분류 추가 보고서
- `타락 개체_860722` — 타락 개체 분류 보고서
- `Cults_871104` — F.H.C 극비 보안 문서
- `불명_Record2_860205` — 피의 호수 부검 기록

### 3. 구역 위험도 문서 연결

`구역 위험도 분류 문서`는 현재 내부 기록 보관서가 아닌 문서 폴더로 남아 있으므로 `docs/Zone_870815/`로 직접 연결한다.

### 4. 검수 모드 연결 ID 표시

일반 모드에서는 기록 제목과 분류만 표시한다.

검수 모드에서는 아래 내부 코드도 함께 표시한다.

- 기록 ID
- 문서 폴더 코드
- standalone 문서 코드

### 5. SELF REVIEW 보강

선택 요소가 있을 때 자체 검수 리포트에 연결 기록 목록을 함께 출력한다.

## 유지 사항

- R24 새 지도 모듈 구조 유지
- R25~R29 시각 보정 유지
- R30 구형 코드 정리 유지
- 세계지도 권역 선택 전용 상태 유지
- 필터 목록 유지
- 검수 모드 유지
- 내부 코드 일반 화면 비노출 유지
- QA 폴더 복구 없음
