# PATCH 5.8.1R-54 — ZoomFiveRestoreAndFinalAuditHotfix

## 목적
R53에서 잘못 적용된 줌 1~10 구조를 제거하고, 이전 기준인 줌 1~5 구조로 복구한다.

## 핵심 수정
- 상세지도 줌 단계를 다시 `줌 1 ~ 줌 5`로 고정
- 마우스 휠 줌 범위도 1~5로 복구
- 줌 2~5 드래그 이동 유지
- R53의 기능별 아이콘 매핑은 유지
- R53의 비존 요소 동적 스케일 보정은 유지
- 존 크기/형태/시각 문법은 유지
- 최대 배율을 10단계로 나누는 방식 제거

## 검수 항목
- 세계지도 권역 선택 전용 유지
- 상세지도 줌 버튼 5개만 표시
- RECORD INDEX 선택 시 필요한 줌은 1~5 범위로만 자동 전환
- 마커/시설/봉쇄선/해양선/링/화살표 스케일 보정 유지
- 클릭 판정 유지
- R53 아이콘 매핑 유지
