# PATCH 5.6.3 — FactionRelationVisualUpgrade

## 기준 버전

`MapPatch5_6_2_ArchiveTransitionCleanup`

## 목표

세력관계도를 단순 트리에서 U.A.C 중심의 360도 관계 추적 행렬로 재구성한다. 세력 간 관계가 협력인지, 감시인지, 적대인지, 불명인지, 비공식 접촉인지 화면에서 바로 구분되도록 한다.

## 포함 내용

- `RELATION TRACE MATRIX` 진입/상태 문구 정리
- U.A.C 중심 방사형 관계도 추가
- 관계선 유형 구분
  - 작전 통제 / 협력
  - 감시
  - 적대
  - 의식 침투
  - 비공식 접촉
- 노드 선택 포커스 모드
  - 선택 노드 강조
  - 직접 연결 노드 중간 강조
  - 무관 노드 흐림 처리
- 관계 분석 패널 강화
  - 선택 세력
  - 상태 태그
  - 직접 연결
  - 분석 설명
  - 관계선 설명
- 관계표와 노드 선택 연동
- 세력별 상태 태그 추가
  - 통제
  - 감시
  - 적대
  - 불명
  - 비공식 접촉
- 노드 클릭 효과음은 임시 유지
- 전체 오디오 재정리는 후속 5.7에서 처리

## 제외 사항

- 지역지도 줌/좌표/레이어 로직 수정 없음
- 기록보관서 구조 수정 없음
- F.H.C / 영상 / 개체 특수 연출 수정 없음
- C.P.D 대피버스, 차량, 장비, 지도 마커는 관계도 노드에서 제외

## 검수 포인트

- 세력관계도 진입 시 `RELATION TRACE MATRIX` 상태가 보이는지
- U.A.C가 중앙에 있고 주변 세력이 방사형으로 배치되는지
- 노드 클릭 시 관련 세력과 관계선만 강조되는지
- 관계 분석 패널 문구가 선택 노드에 맞게 갱신되는지
- 상태 태그가 통제/감시/적대/불명/비공식 접촉으로 표시되는지
- 관계표 행 클릭 또는 노드 클릭 시 강조가 연동되는지
