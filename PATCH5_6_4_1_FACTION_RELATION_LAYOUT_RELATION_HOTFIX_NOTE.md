# PATCH 5.6.4.1 — Faction Relation Layout Relation Hotfix

Base: `MapPatch5_6_4_MenuEntryFXPolish`

## 목적

5.6.4에서 세력관계도가 지나치게 어두워지고, U.A.C 중심 방사형 구조가 거대한 타원 궤도처럼 보이던 문제를 보정한다. 동시에 세력 간 관계선과 설명을 추가해 관계도가 단순 배치도가 아니라 U.A.C 관제망 기준의 관계 추적 화면으로 읽히도록 한다.

## 포함 내용

- 기본 상태에서 모든 세력 노드 가시성 복구
- U.A.C 중심 3단 링 구조 재배치
- 노드 간격과 화면 안쪽 배치 보정
- 관계선 교차 완화 및 곡선/직선 혼합
- 관계선 유형 확장
  - 작전 통제
  - 조사/감시
  - 제한 협력
  - 적대/교전
  - 의식성 침투
  - 비공식 접촉
  - 불명/자료 부족
- 세력 간 관계 추가
- 관계 설명문 보강
- 노드 클릭 시 분석 패널 항목 확장
- 상태 태그 가독성 강화
- C.P.D는 민간 통제국 조직 노드만 유지
- C.P.D 대피버스, 차량, 장비, 지도 마커는 관계도 노드에서 제외 유지

## 검수 포인트

- 기본 상태에서 모든 노드가 보이는지
- 노드 클릭 전부터 과도하게 흐려지지 않는지
- U.A.C를 중심으로 1차 작전권 / 2차 위험권 / 외곽 불명권이 읽히는지
- 관계선이 이전보다 덜 교차하는지
- 관계 분석 패널에 직접 연결, 위험도, U.A.C 판단, 관계 기록이 보이는지
- 관계표가 추가된 관계를 포함하는지
