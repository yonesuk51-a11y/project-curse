# PATCH 5.6 — RecordLinkCodebook + MapReadabilityCleanup

기준 버전: MapPatch5_5_9_MapRuntimeCleanup

## 목적

지도 마커와 기록보관서를 실제로 연결하고, 5.5.7~5.5.9에서 늘어난 지도 정보량을 한 번 더 정리한다. 별도 5.5.9.1 핫픽스로 분리하지 않고 원래 다음 패치였던 5.6에 포함한다.

## 포함 항목

- 마커 상세 패널의 연결 기록을 버튼화
- 연결 기록 클릭 시 기록보관서 내부 기록으로 이동
- 기록보관서 문서 상단에 관련 지도 좌표 패널 추가
- 관련 좌표 클릭 시 지역지도 해당 권역/마커로 이동
- RZ / YZ / GZ / WZ / BLK / GATE / SEA / FAC / ANM / INC 코드북 패널 추가
- 배경 실루엣처럼 보이던 연장 대륙 배경 투명도 완화
- 선택 전 기본 화면에서는 상세 패널 숨김
- 마커 선택 / 중첩 선택 / 개요 상태별 패널 동시 표시 제한
- FEEDS 패널 기본 축소
- 권역 목록과 상세 패널, MARKER STACK 패널이 동시에 과하게 보이는 문제 완화

## 건드리지 않은 것

- 지도 줌/드래그 런타임 대수정 없음
- 세력관계도 구조 변경 없음
- F.H.C / 영상 / 개체 특수 연출 변경 없음
- 전체 효과음 믹싱 변경 없음

## 검수 포인트

- 마커 상세 패널의 연결 기록 버튼이 눌리는지
- 기록보관서 문서에서 관련 좌표 버튼이 보이는지
- 관련 좌표 버튼을 누르면 지역지도 해당 마커로 돌아가는지
- 일반 지도 화면에서 오른쪽 정보가 과하게 겹치지 않는지
- 배경 실루엣/연장 대륙 배경이 이전보다 덜 튀는지
