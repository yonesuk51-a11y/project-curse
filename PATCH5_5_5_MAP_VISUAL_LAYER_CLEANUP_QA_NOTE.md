# PATCH 5.5.5 — MapVisualLayerCleanupQA

기준 버전: `MapPatch5_5_4_MapViewportMarkerStackQA`

## 목적

5.5.4에서 중첩 마커 선택성과 줌 상세 레이어를 보강하면서, 세계/권역 기본 보기에서 보조 시설 블록이 하얀 네모 상자처럼 보이는 문제가 생겼다. 이번 패치는 해당 보조 레이어를 실제 마커와 명확히 분리하고, 줌 단계/필터에 맞는 때에만 표시되도록 정리한다.

## 수정 내용

- ZOOM 1~2에서는 시설/섹터 블록과 도시형 보조 LOD를 숨김.
- ZOOM 4~5에서만 시설/섹터 보조 블록을 낮은 명도와 낮은 투명도로 표시.
- `작전 시설` 필터 선택 시에만 보조 시설 블록을 조금 더 강조.
- 보조 블록 디자인을 더 어둡게 조정해 실제 마커처럼 보이지 않도록 수정.
- 보조 블록에 `HELPER LAYER` 표기를 추가하고, 클릭/포커스 대상에서 제외.
- 상세 줌에서 `VISUAL HELPER LAYER` 안내를 표시해 보조 관제 레이어임을 분명히 함.
- 중첩 마커 선택 패널이 우측 상단 줌 설명 카드와 겹치지 않도록 위치 조정.
- 클러스터 배지의 시각 강도를 약간 낮춰 지도 마커보다 과하게 튀지 않도록 조정.
- 검수 모드에 `VISUAL LAYER QA` 패널을 추가해 권역/줌/필터/보조 레이어 상태를 확인 가능하게 함.
