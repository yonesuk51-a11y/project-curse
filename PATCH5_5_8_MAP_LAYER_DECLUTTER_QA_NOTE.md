# PATCH 5.5.8 — MapLayerDeclutterQA

기준 버전: `MapPatch5_5_7_MapTacticalVisualUpgrade`

## 목적

5.5.7에서 추가된 해칭 오염권, 감시 반경, FEEDS 패널, 위성/드론 감시선, 미니 라벨이 ZOOM 5에서 한꺼번에 겹쳐 보이는 문제를 정리한다.

## 변경 사항

- 줌 단계별 전술 레이어 표시 규칙 적용
- ZOOM 1: 대형 해칭 오염권만 낮은 명도로 유지, 반경/링크/라벨/시설 블록 숨김
- ZOOM 2: 요약 감시 반경만 낮은 명도로 표시, 미니 카드/시설/링크 숨김
- ZOOM 3: 전술 관제 단계, 라벨은 선택 마커 중심으로 제한
- ZOOM 4: 시설 관제 단계, 비선택 카드와 링크를 제한
- ZOOM 5: 기록 분석 단계, 모든 레이어 동시 표시 금지. 선택 마커 관련 반경/링크/라벨만 표시
- 마커 선택 시 선택 코드와 연결된 전술 레이어만 강조하는 포커스 모드 추가
- 비선택 해칭/반경은 낮은 투명도로 배경화
- `HELPER LAYER`, `FAC-A/FAC-B`, 분석 레이블 등 보조/검수 문구는 일반 모드에서 숨김
- 검수 모드 전용 `LAYER VISIBILITY QA` 패널 추가

## 제외 사항

- 지도 조작 로직 대수정 없음
- 좌표 대규모 재배치 없음
- 세력관계도/기록보관서/F.H.C/영상/개체 연출 변경 없음
- 전체 효과음 믹싱 변경 없음

## 검수 포인트

- ZOOM 5에서 모든 라벨, 반경, 링크가 동시에 켜지지 않는지 확인
- 마커를 선택했을 때 선택 대상 주변만 강조되는지 확인
- 비선택 구역은 배경처럼 낮은 명도로 보이는지 확인
- 일반 모드에서 `HELPER LAYER`, `FAC-B` 같은 문구가 보이지 않는지 확인
- 검수 모드에서만 레이어 표시 QA 정보가 보이는지 확인
