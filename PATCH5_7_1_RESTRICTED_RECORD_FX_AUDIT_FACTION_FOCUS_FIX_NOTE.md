# PATCH 5.7.1 — RestrictedRecordFXAudit + FactionFocusFix

## 기준 버전
MapPatch5_7_AudioFXPolish

## 목적
F.H.C / 영상 기록 / 개체 기록의 특수 연출이 지정된 기록군에만 유지되는지 회귀 검수하고, 세력관계도에서 U.A.C 선택 시 직접 연결 노드가 오히려 흐려지거나 사라져 보이는 포커스 로직 문제를 수정한다.

## 포함 내용

### 1. 세력관계도 포커스 로직 수정
- U.A.C, UAC, uac 등 내부 ID 매칭을 통일한다.
- U.A.C 선택 시 직접 연결 노드가 반드시 선명하게 보이도록 보정한다.
- 선택 노드 / 직접 연결 노드 / 2차 연결 노드 / 무관 노드의 밝기 규칙을 다시 정의한다.
- 무관 노드도 완전 숨김 처리하지 않고 최소 윤곽, 이름, 상태 태그를 유지한다.
- 기존 5.6.3, 5.6.4, 5.6.4.1, 5.7 관계도 렌더링을 덮어쓰는 5.7.1 전용 관계도 렌더를 추가한다.

### 2. U.A.C 직접 연결 노드 강제 표시
U.A.C 선택 시 아래 노드는 직접 연결 노드로 강조된다.

- N.H.C
- S.I.D
- F.H.C
- C.P.D
- A.R.F
- Ushnoda Cult
- Haimun

아래 노드는 2차 또는 외곽 관계로 중간 명도 상태를 유지한다.

- Ash Crew
- Syndicate
- Amarion

### 3. 관계 분석 패널 보강
- 직접 연결과 2차 연결을 분리 표시한다.
- 관계 기록에는 선택 세력과 직접 연결된 관계 설명이 이어서 표시된다.
- 관계표 클릭 시 해당 관계의 시작 노드 기준으로 포커스가 이동한다.

### 4. Restricted Record FX Audit
- F.H.C 검열 연출은 F.H.C 기록군에만 유지한다.
- 손상 영상 피드 연출은 영상 기록군에만 유지한다.
- 생체 스캔 연출은 개체/부검/유전자 기록군에만 유지한다.
- 일반 기록에는 특수 연출이 번지지 않도록 분류 표시를 보강한다.

## 검수 포인트
- 세력관계도 진입 시 모든 세력 노드가 보이는지 확인한다.
- U.A.C 클릭 시 N.H.C, S.I.D, F.H.C, C.P.D, A.R.F, Ushnoda Cult, Haimun이 직접 연결 노드로 강조되는지 확인한다.
- 무관 노드가 완전히 사라지지 않고 낮은 명도로 남는지 확인한다.
- F.H.C, 영상, 개체 특수 연출이 해당 기록군에만 적용되는지 확인한다.
- 일반 기록보관서 문서에 F.H.C/영상/개체 전용 연출이 번지지 않는지 확인한다.
