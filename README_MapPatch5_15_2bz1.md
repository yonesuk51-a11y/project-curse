# Project Curse U.A.C Server — 5.15.2bz1

2bz의 장비 체계 병합 내용은 유지하고, 장비 분류 QA가 제거된 구형 장비 코덱스를 필수 화면으로 오인하던 문제를 수정한 핫픽스입니다.

핵심 수정:
- 기록보관실의 6개 통합 문서를 장비 분류의 기준으로 사용
- InterfaceReset에서 제거된 구형 Equipment Codex는 QA 필수 대상에서 제외
- 구형 코덱스가 실제로 존재할 때만 내부 필터를 추가 검사
- 숨겨진 기록 본문도 실패·철수·회수·폐기 용어 검사에 포함
- `screenSweep()`의 상단 ok/error/warning 수치와 세부 검사 결과를 일치시킴
- `codexState` 진단값 추가

콘텐츠와 화면 구성은 5.15.2bz에서 변경하지 않았습니다.
