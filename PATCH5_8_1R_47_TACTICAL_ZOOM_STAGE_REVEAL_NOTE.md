# PATCH5_8_1R_47_TACTICAL_ZOOM_STAGE_REVEAL_NOTE

## 목적

R45~R46의 `요약 / 기본 / 확대 / 정밀` 보기 체계를 제거하고, 상세 권역지도 표시 방식을 `줌 1 ~ 줌 5` 단계형 전술 지도 방식으로 재구성한다.

## 주요 변경

- `요약` 보기 제거
- `정밀` 보기 제거
- 정밀 전용 데이터 제거
- 상세지도 줌 단계 재구성
  - 줌 1: 존 중심, 나머지는 점 표시
  - 줌 2: 주요 시설/사건 표시
  - 줌 3: 봉쇄선/봉쇄 거점/현상 기록 표시
  - 줌 4: 보조 시설/통신권 표시
  - 줌 5: 최대 라벨/관계권 중심/근접 후보 확인
- 세계지도는 권역 선택 전용 상태 유지
- 기존 지도↔문서 연결, 색인, 검수 기능 유지

## 데이터 변경

- `detail: precision` 요소 16개 제거
- 기존 `detail: zoom` 요소 16개를 `zoomMin: 4`로 전환
- 전체 요소 수: 181개 → 165개
- `ZOOM_STAGE_LAYER` 메타데이터 추가

## 검수

- JS 문법 체크 완료
- 데이터 모듈 문법 체크 완료
- 중복 ID 없음
- 정밀 전용 데이터 잔존 없음
- 세계지도 줌 비활성 유지
