# Patch 5.1 — F.H.C Restricted Redaction

기준: MapPatch5_SafeUI.

## 목적
- 안정화된 5차 기능은 유지한다.
- F.H.C 극비 보안 문서에만 제한 기록 UI를 좁게 적용한다.
- 이전에 문제였던 빈 공간 검열 사각형과 본문 중간 대각선 장식 남발을 방지한다.

## 적용
- F.H.C 문서 카드에 F.H.C SEALED / TRACE ENABLED 칩 추가.
- F.H.C 문서 열람 시 내부 접근 오버레이 표시.
- F.H.C 문서 본문 상단에 보안 패널 추가.
- 실제 텍스트 내부에만 짧은 인라인 검열 표시 삽입.
- 일반 문서, 지도, 세력정보, 관계도, 배경음, 기본 메뉴 기능은 건드리지 않음.

## 제한
- F.H.C 본격 검열 효과의 1차 안전 적용판.
- 영상 손상 피드, 개체 생체 스캔, 세력 관계도 고급 효과, 지도 상세 레이어는 미적용.
