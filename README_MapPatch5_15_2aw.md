## 핵심 변경
- 아카이브 공개 카드와 문서 제목을 **"종교"** 로 변경.
- Cults_871104 시퀀스를 종교/오염 흐름 중심으로 재구성.
- 신규 내용 반영:
  - Signs of Corruption → `타락의 과정 및 형태`
  - Hina's Ignorance → `침묵성 타락`
  - Hina's Stare → `융합성 타락`
  - Hina's Suffering → `변조 경고`
- 타락교 / 혈교 / 혈액 이동 경로 조정 / 혈무의 제작 과정은 **오른쪽 보조 이미지형** 레이아웃으로 조정.
- 신원불명의 신자 / 가면을 쓴 존재 / 타락의 과정 및 형태 / 침묵성 타락 / 융합성 타락 / 피의 호수를 거니는 자들 / 혈액 저장소 / 제압된 타락체는 **대형 증거사진형** 레이아웃으로 조정.
- 시퀀스 설명줄이 이전처럼 천천히 미끄러지지 않고, 더 짧은 **팟/글리치형 등장**으로 보이게 조정.
- 증거사진 프레임의 불필요한 패딩을 줄이고, 이미지와 텍스트 사이 간격을 압축.
