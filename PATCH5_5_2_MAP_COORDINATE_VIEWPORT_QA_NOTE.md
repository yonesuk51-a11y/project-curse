# PATCH 5.5.2 — MapCoordinateViewportQA

## 목적
5.5.1에서 마커 상세 패널은 복구됐지만, 일부 권역에서 지명과 마커 위치가 어긋나거나 지도 이미지가 잘려 보이는 문제가 남아 있었다. 이번 패치는 동아시아 한정 보정이 아니라 전체 권역 지도 좌표 신뢰도를 다시 잡는 QA 패치다.

## 적용 범위
- 전체 권역 베이스맵을 고정 16:9 관제면으로 재생성
- 세계 / 동아시아 / 유럽 / 북미 / 남아시아 / 동남아·인도양 / 오세아니아 / 중동 / 아프리카 좌표면 재정렬
- 지도 이미지와 SVG 마커 레이어가 같은 권역 범위를 쓰도록 유지
- 부산 해상 통제 허브를 한반도 남동부 연안 좌표로 보정
- 해상 / 항만 / 육상 표면 판정 값을 마커 데이터에 보강
- 마커 상세 패널에 권역, 표면, 위치 판정, 좌표 상태 추가
- 검수 모드에서 현재 마커의 권역 / 표면 / 좌표 상태를 확인하는 MARKER QA 패널 추가

## 의도적으로 건드리지 않은 것
- 세력관계도 구조
- 기록보관서 연출
- F.H.C 검열 연출
- 영상 손상 피드
- 개체 생체 스캔
- 전체 효과음 믹싱

## 검수 포인트
1. 동아시아에서 부산 관련 표식이 일본 열도 위에 붙어 보이지 않는지 확인한다.
2. 세계지도와 권역지도에서 대륙이 잘린 것처럼 보이는지 확인한다.
3. 해상 마커와 육상 마커의 상세 패널 표면 판정이 설명과 맞는지 확인한다.
4. 검수 모드에서 MARKER QA 패널이 활성화되는지 확인한다.
5. 마커 클릭 시 오른쪽 아래 상세 패널이 계속 유지되는지 확인한다.
