# PATCH 5.4 — Faction Relation Trace

기준: MapPatch5_3_BioScanReport

## 적용 범위
- 세력 관계도 페이지
- 세력 정보 카드 클릭 보조 연출
- 기록면 탭 전환 보조 연출

## 반영
- 세력 관계도 노드 클릭/키보드 선택 지원
- 선택 세력, 관련 경로, 관계표 행 강조
- FACTION TRACE 분석 패널 추가
- 관계 상태 / 위협 지수 / 접촉 기록 표시
- 기존 오디오 파일을 건드리지 않고 WebAudio 기반 짧은 UI 펄스 효과음 추가
- 기록면 탭 전환 시 짧은 RECORD SURFACE REMAP 연출 추가

## 건드리지 않음
- 지도 확대/축소/대륙 전환
- 기록 열람 기본 함수
- F.H.C 검열 효과
- 영상 피드 효과
- 개체 생체 스캔 효과
- 기존 배경음 파일
