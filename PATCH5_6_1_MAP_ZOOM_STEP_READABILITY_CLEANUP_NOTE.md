# PATCH 5.6.1 — MapZoomStepReadabilityCleanup

기준 버전: `MapPatch5_6_RecordLinkCodebook`

## 목표

지도 줌을 1% 단위 자유 배율에서 `ZOOM 1~5` 고정 단계식으로 정리하고, 확대 화면에서 보이던 배경 실루엣/잔상 레이어를 일반 모드에서 제거해 지도 가독성과 조작 안정성을 높인다.

## 적용 내용

### 1. 5단계 고정 줌

지도 줌 배율을 다음 단계로 고정했다.

- ZOOM 1: 100% / 권역 보기
- ZOOM 2: 140% / 작전권 보기
- ZOOM 3: 185% / 전술 관제
- ZOOM 4: 240% / 시설 관제
- ZOOM 5: 300% / 기록 분석

`+ / -` 버튼은 이제 1%식 증감이 아니라 다음/이전 단계로 이동한다.

### 2. 휠 줌 단계화

휠 줌도 연속 배율 계산 대신 입력량을 모아 한 단계씩 이동한다. 이로 인해 줌 중간값마다 LOD/클러스터/레이어 규칙이 반복 갱신되는 일을 줄인다.

### 3. 최대 줌 300% 제한

기존 최대 320% 계열을 300%로 제한했다. ZOOM 5가 기록 분석의 최종 단계가 된다.

### 4. 배경 실루엣/잔상 레이어 정리

빈공간 완화용 연장 대륙 배경은 일반 모드에서 숨기고, 검수 모드에서만 희미하게 확인 가능하게 했다.

일반 모드에서 숨긴 것:

- 연장 대륙 배경
- HELPER/FAC 계열 보조 라벨
- QA/검수 패널 잔류 라벨
- 낮은 줌 단계의 보조 LOD 선/점/라벨

### 5. 전술 시각 레이어 저명도화

오염 해칭, 감시 반경, 링크선, 미니 카드, 시설 셀은 선택 마커가 없을 때 훨씬 낮은 명도로 제한된다. 선택 마커가 있을 때만 관련 레이어를 강조한다.

### 6. FEEDS 패널 축소

기본 상태의 `ACTIVE FEEDS` 패널을 작은 상태 칩에 가깝게 축소했다. 지도 본문과 오른쪽 정보 패널을 과하게 가리지 않도록 조정했다.

## 건드리지 않은 것

- 지도 마커 데이터
- 지도 좌표 투영
- 기록보관서 연결 구조
- 세력관계도
- F.H.C / 영상 / 개체 연출
- 전체 효과음 믹싱

## 검수 포인트

- `+ / -` 버튼이 ZOOM 1~5 단계로만 이동하는지
- 휠 줌이 1% 단위로 흔들리지 않는지
- 최대 줌이 300%로 제한되는지
- 일반 모드에서 배경 실루엣이 사라졌는지
- 마커 선택 전 정보량이 줄었는지
- 선택 마커 관련 레이어만 강조되는지
