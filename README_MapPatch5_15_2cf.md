# MapPatch 5.15.2cf — Archive Consolidation and Screen Presentation Pass

기준 빌드: `5.15.2ce Organization Canon Reconciliation Pass`

## 현재 기록 복원 상태

`구역 위험도 분류 문서`, `레드존 이상현상 및 오염 기준 문서`, `N.H.C 현장 작전·장비·봉쇄 규정 문서`, `괴이`는 사용자가 제공한 ENEX 원문을 기준으로 복원됐다. 짧은 임시 요약 대신 표·세부 절차·원본 이미지를 포함하는 독립 장문 페이지로 열리며, 현재 확정된 U.A.C·N.H.C·S.I.D·Ash Crew 연혁에 맞춰 충돌하는 옛 명칭과 시점을 정리했다.

`괴이`는 Ferals·Superiors·Unusuals의 원문 체계를 유지하면서 확인 개체 12종을 개별 항목으로 복원했다. Celestials·Odious는 우시노다교 계급이 아닌 과거 현장 하위분류임을 명시하고, 부적은 CI-O 지원 물체로 재분류했다.

열람 방식은 종교 기록과 같은 전체화면 시퀀스다. 기록보관소의 `괴이`를 누르면 별도 페이지로 이동하지 않고 사용자 입력을 이어 받아 즉시 재생한다. 종교의 원본 손상 신호 영상을 공통 도입으로 사용하고, VHS 전환·효과음·라디오 정적음도 공유한다. 괴이의 전용 배경음은 `pc5152cf_feral_dying_memories_bgm.mp3`에서 루프 재생하며 메뉴 배경음은 시퀀스 동안 정지한다. 작성 경위와 네 개 분류 장을 포함한 총 24면이며 이전·다음·일시정지·처음부터 조작을 제공한다.

종교·불멸을 향해·괴이를 제외한 일반 설정글은 왼쪽 고정 목차와 오른쪽 본문으로 구성된다. 첫 문단, 후속 문단, 표, 회수 이미지와 하위 항목의 간격을 공통 규칙으로 정리했으며, 기록보관소 복귀 시에는 최초 부팅 화면을 다시 재생하지 않는다.

이번 패치는 기록 수를 늘리는 대신 같은 주제의 문서를 대표 기록철 안으로 병합한다. 목록에서는 무엇을 열어야 하는지 바로 보이고, 상세에서는 필요한 세부 기록만 펼쳐 읽는 구조를 목표로 한다.

## 대표 기록철

1. 사건·회수 기록철 — 피의 호수 이후 사건, 회수 자료, 이탈·유전자 기록
2. 세력·인물 기록철 — U.A.C, N.H.C, S.I.D, F.H.C, 우시노다교, 신디케이트·아마리온
3. 이상현상·개체 기록철 — 레드존, 시간·통신 오염, Blood Gate, 개체 분류
4. 작전·장비·규정 기록철 — 차폐, 신호, 혈성 오염, 회수·소각, 봉쇄, 제압 체계

각 기록철의 세부 내용은 접히는 기록면으로 유지한다. 같은 내용을 다시 보여 주는 연결 카드나 태그 버튼은 사용하지 않는다.

## 잠금 범위

`종교`와 `불멸을 향해`는 대표 기록철의 본문으로 복사하거나 재배치하지 않는다. 각 기록철의 `원본 기록 열람` 버튼은 기존 원본 기록으로 직접 연결하며, 본문·이미지·영상·효과·음향·레이아웃·반응형 동작은 변경하지 않는다.
