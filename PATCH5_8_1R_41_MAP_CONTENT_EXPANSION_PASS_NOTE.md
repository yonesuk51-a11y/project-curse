# PATCH 5.8.1R-41 — MapContentExpansionPass

## 목적

R40에서 분리한 지도 데이터 모듈 구조를 유지한 상태로, 지역지도 콘텐츠를 데이터 중심으로 확장한다. 렌더러 본체에 기능을 덧붙이지 않고 `assets/data` 모듈 위주로 사건·현상·시설·봉쇄·보조 존을 보강하는 것이 목적이다.

## 적용 범위

수정 파일:

- `assets/data/uac_region_map_data.js`
- `assets/data/uac_region_summaries.js`
- `assets/js/r24_regional_map_rebuild.js`
- `README.md`
- `PATCH5_8_1R_41_MAP_CONTENT_EXPANSION_PASS_NOTE.md`

## 콘텐츠 확장 원칙

- 새 존 체계는 추가하지 않는다.
- 존 명칭은 기존 5종만 유지한다.
  - 그린존
  - 옐로우존
  - 레드존
  - 블랙존
  - 화이트존
- 세계지도에는 데이터 마커를 추가하지 않는다.
- 상세지도 권역 내부에서만 사건·현상·시설·봉쇄 요소를 확장한다.
- 관계권이 없는 임의 마커 추가를 피하고, 기존 사건권/작전권/봉쇄권/항만권/후방권에 맞춰 보강한다.
- 봉쇄선을 추가할 경우 같은 관계권에 봉쇄 거점을 함께 유지한다.
- 블랙존은 추가하지 않고 기존 희소성을 유지한다.
