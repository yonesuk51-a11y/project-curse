# PATCH 5.8.1R-32 — OverlapSelectionLayerFix

## 목적
R31 이후 지도 요소가 겹쳐 있을 때 한 요소를 선택하면 다른 요소가 사라져 보이거나, 뒤에 깔린 요소를 다시 선택하기 어려운 문제를 수정한다.

## 변경 사항
- 선택 상태에서 비선택 요소를 완전히 숨기지 않고 불투명도만 낮추도록 조정.
- 같은 관계권 요소는 더 높은 불투명도로 유지하여 작전권 묶음이 계속 읽히게 조정.
- 흐려진 요소도 `pointer-events`를 유지하여 계속 선택 가능하게 보정.
- 지도 클릭 지점 주변의 겹친 후보를 계산하는 선택 후보 메뉴 추가.
- 후보 메뉴에서 레드존/시설/사건좌표/봉쇄 거점/현상 기록 등 겹친 요소 중 원하는 대상을 직접 선택 가능.
- 선택 요소가 점/선 계열인 경우 최상단으로 이동하되, 존 선택 시에는 다른 마커를 덮어버리지 않도록 존은 원래 레이어 질서를 유지.
- 자체 검수 리포트에 겹친 선택 후보 목록을 표시하도록 보강.

## 유지 사항
- R24 새 지도 모듈 구조 유지.
- R25~R29 시각 보정 유지.
- R30 패키지 정리 유지.
- R31 지도 → 문서 연결 유지.
- 세계지도 권역 선택 전용 상태 유지.
- 검수 모드와 일반 모드 분리 유지.
