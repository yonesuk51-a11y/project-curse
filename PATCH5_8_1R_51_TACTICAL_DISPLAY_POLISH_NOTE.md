# PATCH5_8_1R_51_TACTICAL_DISPLAY_POLISH_NOTE

## 목적

R50의 조작 안정화 구조를 유지하면서 U.A.C 지역지도의 표시 방식을 더 전술 지도처럼 읽기 쉽게 다듬는다.

## 변경 사항

- 핵심/주요/표준/보조 지도 요소 위계 클래스 추가
- 선택 요소의 관계권을 전술 묶음 프레임으로 표시
- 줌 4 이상에서 관계권 내부 연결선 표시
- 줌 3 이상에서 선택 요소 미니 정보 카드 표시
- 봉쇄선 끝점 노드와 방향 표시 강화
- 우측 컨트롤 경로 표시 추가
- 전술 표시 검수 API 추가

## 유지 사항

- 세계지도 권역 선택 전용 유지
- 줌 1~5 단계 유지
- 마우스 휠 줌 유지
- 줌 2~5 드래그 이동 유지
- 선택 시 자동 중심 복귀 없음 유지
- 새 존 체계 추가 없음
- 블랙존 추가 없음
