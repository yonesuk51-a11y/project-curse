# MapPatch 5.15.2ce — Organization Canon Reconciliation Pass

기준 빌드: `5.15.2ca1 TerminalHome AmbientScopeFix`

이번 패치는 화면 재장식 전에 설정과 동작의 소유권을 먼저 정리한다. 같은 조직이 화면마다 다른 이름·분류·관계로 보이던 문제, 태그 버튼과 연결 기록이 반복되던 문제, 여러 이벤트가 같은 효과음을 겹쳐 재생하던 문제를 한 기준으로 묶었다.

- U.A.C: `Urban Anomaly Containment / 도시 이상현상 격리국`
- 하이문: 교단이 아닌 도시 기반 범죄조직
- F.H.C: 기업국가급 세계 영향력 1위
- 아마리온: 세계 영향력 2위이며 F.H.C의 직접 경쟁 세력
- N.H.C: 과거 U.A.C 산하, 현재 독립 조직이자 특수 지원 협정 대상
- Ash Crew: 독립 세력이 아닌 N.H.C 산하 오염처리부대
- 신디케이트: 반 U.A.C 연합, 현재 F.H.C의 암묵적 지원, 우시노다교와는 제한적 임시 동맹
- 우시노다교: 타락교·혈교·그림자교 3파벌, 파벌당 로드 1명·사도 4명
- 센티널: 계급이 아니라 로드 귀속 호위·전투·처형 자산
- A.R.F: 상위 소속 축으로만 확정. 연결 대상은 미확정이므로 관계선 보류

## 화면 정리

- 기록보관소·세력 파일·관계도에 반복되던 태그 버튼 줄을 숨겼다.
- 자동 생성된 관련 기록·연결 기록 패널 가운데 본문과 같은 정보를 반복하는 요소를 제거했다.
- 기록 검색창은 남기고 전체 폭으로 넓혔다.
- 활성 세력 파일과 관계도는 동일한 정사 레지스트리를 읽는다.

## 오디오

- 메뉴 환경음, 페이지 이동, 서랍, 기록 장착, 페이지 전환, 접근 거부를 중앙 상태관리자가 소유한다.
- hover 효과음은 사용하지 않는다.
- 한 입력에 여러 구형 이벤트가 호출되어도 중앙 cooldown과 음소거 규칙으로 중복 재생을 차단한다.
- 기록 화면에서는 메뉴 환경음을 정지하고 기록 전용 오디오는 별도로 유지한다.
