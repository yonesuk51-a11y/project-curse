# PATCH 5.8.1R-53 — MarkerScaleAndIconMappingRebuild

## 목적

존 크기는 유지하고, 존을 제외한 시설·기지·사건·통신·봉쇄선·해양선·링·화살표의 시각 크기와 아이콘 문법을 재정리한다.

## 적용 사항

- 상세지도 줌 단계를 줌 1~10으로 확장.
- 마우스 휠 줌 범위를 1~10으로 확장.
- 존은 기존 불규칙 오염권 크기/색상 유지.
- 비존 요소는 카메라 배율과 별도 계산된 시각 스케일을 적용.
- 시설/마커/기지 아이콘을 기능별 픽토그램으로 자동 매칭.
- 항만/공항/보급/관측/통신/검문/해상부표/사건/경고/오염반응/블랙존 징후 아이콘 분리.
- 봉쇄선·해양선 두께, 끝점 노드, 방향 화살표, 감시 링 크기를 축소.
- 클릭 판정용 투명 타겟은 유지하여 조작성을 보존.
- RECORD INDEX와 선택 정보에 아이콘 매칭 정보를 반영.

## 유지 사항

- 세계지도 권역 선택 전용 유지.
- 세계지도 줌/드래그/FIT/WIDE 없음.
- 일반 선택 시 자동 중심 복귀 없음.
- 일반 화면 라벨 최소화 / 검수 모드 세부 정보 유지.
