# PATCH 5.5.6 — MapPerformanceMarkerFineTuneQA

## 기준 버전

`MapPatch5_5_5_MapVisualLayerCleanupQA`

## 목표

지역지도에서 줌인/줌아웃 및 드래그 조작 중 발생하는 렉 체감을 줄이고, 마커 좌표/표면 QA 정보를 계속 확인할 수 있도록 보강한다.

이번 패치는 시각효과 추가보다 **성능 안정화**를 우선한다. 세력관계도, 기록보관서, F.H.C, 영상 피드, 개체 스캔, 전체 오디오 믹싱은 변경하지 않는다.

## 적용 내용

### 1. 줌 HUD/로딩 체감 축소

- 줌인/줌아웃 시 매번 크게 뜨던 `TACTICAL LAYER SYNC` 계열 HUD 호출을 줄였다.
- 기존 강제 리플로우(`offsetWidth`) 방식의 HUD 재시작을 `requestAnimationFrame` 방식으로 변경했다.
- 줌 조작 중에는 큰 지도 로딩/동기화 오버레이가 체감 렉처럼 보이지 않도록 축소 표시한다.

### 2. 드래그 transform 최적화

- 포인터 이동마다 즉시 전체 transform을 재동기화하지 않고, `requestAnimationFrame` 기준으로 한 번씩만 반영한다.
- 지도 이미지, SVG 마커 레이어, 전술 LOD 레이어는 기존처럼 같은 transform 상태를 유지한다.

### 3. 마커 클러스터 재계산 디바운스

- 5.5.4의 중첩 마커 클러스터 계산이 줌/드래그 중 과도하게 반복되지 않도록 보정했다.
- `style` 속성 변화 감시를 제거하고, 실제 줌 단계/class 변화 위주로만 갱신한다.
- 휠 줌 중에는 클러스터 재계산을 지연 처리한다.

### 4. 드래그/줌 중 상세 레이어 일시 정지

`map-fast-nav` 상태가 켜진 동안 다음 레이어를 일시적으로 숨긴다.

- 전술 LOD 레이어
- 도시형 LOD 레이어
- 클러스터 배지 레이어
- 보조 시설/섹터 설명 레이어
- 일부 라벨/디버그 라벨

조작이 끝나면 다시 복구된다.

### 5. 연장 대륙 배경 최적화

- 줌/드래그 중에는 연장 대륙 배경의 필터 부담을 낮췄다.
- 검은 빈공간 완화 기능은 유지하되, 조작 중에는 밝기/채도/필터를 낮춰 렌더링 부담을 줄인다.

### 6. 지도 성능 QA 패널 추가

검수 모드에서 `MAP PERFORMANCE QA` 패널을 추가했다.

표시 항목:

- 현재 권역
- 현재 줌 단계
- 마커 수
- 확인 완료 마커 수
- 재검수 필요 마커 수
- 현재 렌더 상태
- 최근 조작 상태

### 7. 좌표 검수 크로스헤어 추가

검수 모드에서 중앙 좌표 크로스헤어를 표시한다.

목적:

- 현재 지도 중심 기준 확인
- FIT/WIDE 후 중심 복귀 확인
- 좌표 보정 패치에서 기준점 확인

### 8. 마커 QA 메타데이터 유지

- 각 마커에 `data-pc556-qa`, `data-pc556-surface`, `data-pc556-geo` 보조 정보를 부여한다.
- 기존 오른쪽 아래 마커 상세 패널, 중첩 마커 선택 목록, 좌표 상태 표시는 유지한다.

## 이번 패치에서 의도적으로 제외한 것

- 전체 마커 대규모 재배치
- 전술 지도 시각효과 대규모 추가
- 세력관계도 360도 개편
- 기록보관서 전환 연출 대수정
- F.H.C / 영상 / 개체 특수 연출 수정
- 전체 효과음 믹싱
