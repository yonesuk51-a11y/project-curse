# PATCH 5.8.1R-29 — RegionalMapVisualPolish

## 목적

R24~R28에서 안정화한 새 지역지도 모듈을 기준으로, 데이터 구조나 필터 체계를 추가로 바꾸지 않고 실제 화면의 시각 완성도를 정리한 패치입니다.

## 유지 기준

- R24 이후 단일 데이터·단일 렌더러·단일 필터 구조 유지
- R27 관계권 보정 데이터 유지
- R28 구형 지도 코드 정리 상태 유지
- 세계지도는 권역 선택 전용 화면 유지
- 세계지도에 존/시설/사건/봉쇄선/통신권 표시 금지
- 검수 모드는 일반 필터가 아닌 내부 코드 라벨 표시 상태로 유지
- 일반 화면에서는 내부 코드 라벨 비노출 유지

## 주요 변경점

### 1. 세계 권역 선택 화면 마감

- 권역 버튼 위치를 대륙 중심에 더 맞게 소폭 보정했습니다.
- 세계지도 크롭과 밝기/채도/대비를 조정해 버튼과 지도 윤곽이 과하게 튀지 않도록 했습니다.
- 권역 버튼 크기와 그림자를 줄여 첫 화면이 덜 답답하게 보이도록 정리했습니다.

### 2. 존 표현 시각 보정

- 레드존에 낮은 농도의 외곽 확산층을 추가해 단순 원형이 아니라 오염이 번진 느낌을 강화했습니다.
- 레드존 중심부와 꼬리 레이어의 채도를 낮춰 과한 경고판 느낌을 줄였습니다.
- 옐로우존은 레드존처럼 보이지 않도록 채도와 발광감을 낮추고 완충 감시권에 가깝게 조정했습니다.
- 블랙존은 더 어두운 심부 코어처럼 보이도록 채도와 균열 표시를 억제했습니다.
- 그린존/화이트존은 오염권이 아니라 후방 운영권/검문 관리권처럼 보이도록 관리 링을 추가했습니다.

### 3. 봉쇄선·감시선 보정

- 육상 봉쇄선, 항만 차단선, 해상 감시선, 사막 감시선의 선형 표현을 다시 분리했습니다.
- 항만 차단선은 짧고 단단하게 보이도록 선 두께와 점선을 줄였습니다.
- 해상 감시선은 넓고 흐린 감시권처럼 보이도록 투명도와 점선 간격을 완화했습니다.
- 사막 감시선은 시설선처럼 보이지 않도록 흐리고 긴 감시선 톤으로 조정했습니다.

### 4. 마커 가독성 정리

- 작전 시설, 봉쇄 거점, 사건 좌표, 현상 기록, 통신권의 시각 차이를 조금 더 명확히 했습니다.
- 시설/기지 아이콘 자체는 크게 키우지 않고 선명도와 대비만 제한적으로 조정했습니다.
- 선택 강조와 관계권 강조 효과를 줄여 일반 지도 화면을 덜 과장되게 만들었습니다.

### 5. 우측 패널 부담 완화

- 우측 패널 폭과 여백을 줄여 지도 영역을 조금 더 크게 보이게 했습니다.
- SELF REVIEW 패널은 유지하되 높이와 글자 크기를 줄여 일반 관람 화면에서 덜 튀게 했습니다.
- 검수 리포트 텍스트 영역은 계속 사용할 수 있으나 기본 높이를 줄였습니다.
