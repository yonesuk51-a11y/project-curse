# PATCH 5.5.1 — TacticalMapQA

기준 버전: `MapPatch5_5_TacticalMapLayers`

## 목적

5.5에서 추가된 지역지도 전술 레이어를 유지하되, 검수 중 확인된 지도 빈공간 노출, 부자연스러운 설명문, 지도 효과음, 마커 상세 패널 누락, 기록면 스캔라인 중복 문제를 보정한다.

## 적용 범위

- 지역지도 뷰포트 클램프 보정
- WIDE/FIT/줌아웃 시 지도 외부 빈공간 노출 방지
- 드래그 한계값을 지도 프레임 기준으로 재계산
- 지도 이미지, SVG 마커, 전술 LOD 레이어 transform 동기화
- 확대 시 마커/라벨 역스케일 보정
- 동아시아 권역 좌표 기준 1차 보정
  - `east` 권역 위도 범위를 실제 SVG 생성 기준인 `-5 ~ 58`로 정렬
- 지도 설명문 수정
  - 외부 작품명 언급 제거
  - 1600×900/SVG viewBox 같은 메타 표현을 U.A.C 내부 관제 문구로 대체
- 지역지도 로딩 연출 변경
  - 코드 출력형 로딩 제거
  - `REGIONAL SURFACE LINK / 권역 관제면 연결` 연출로 교체
- 지도 효과음 완화
  - 고주파/찍찍거림을 줄이고 낮은 sine + lowpass 기반으로 변경
  - 줌/필터 연속 입력 중복 재생 완화
- 마커 클릭 상세 패널 복구
  - 우측 하단 카드에 코드, 시각 표현, 위치 상태, 역할, 표면, 관측, 동기화, 연결 기록 표시
- 기록면 탭 전환 스캔라인 중복 제거
  - 중앙/상단 스캔라인이 겹치지 않도록 중간 선 제거
- 왼쪽 메뉴별 진입 연출 분리
  - 세계 역사, 세력 정보, 세력 관계도, 지역 지도, 기록 보관서 각각 다른 진입 문구 사용

## 건드리지 않은 것

- F.H.C 검열 로직
- 영상 손상 피드 로직
- 개체 생체 스캔 로직
- 세력관계도 노드 구조
- 기록 문서 본문
- 지도 마커 데이터의 대규모 재작성

## 검수 포인트

1. 지역지도에서 WIDE/FIT/줌아웃 후 빈공간이 보이지 않는지 확인.
2. 확대 상태에서 드래그할 때 지도 밖 영역이 노출되지 않는지 확인.
3. 동아시아에서 부산, 홍콩, 란저우, 일본 서남부 표식이 심하게 어긋나지 않는지 확인.
4. 마커 클릭 시 오른쪽 아래 상세 패널이 서버식 카드로 표시되는지 확인.
5. 기록면 탭 클릭 시 스캔라인이 중복으로 보이지 않는지 확인.
6. 지역지도 로딩이 코드 출력이 아니라 권역 관제면 연결 연출로 나오는지 확인.
7. 줌인/줌아웃 효과음이 날카롭게 튀지 않는지 확인.
