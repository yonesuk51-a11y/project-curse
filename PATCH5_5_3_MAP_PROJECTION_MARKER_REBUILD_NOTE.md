# PATCH 5.5.3 — MapProjectionMarkerRebuild

기준 버전: MapPatch5_5_2_MapCoordinateViewportQA

## 목표

세계지도부터 아프리카까지 남아 있던 좌표 불신 문제를 줄이기 위해, 지역지도 마커를 픽셀 감각이 아니라 권역별 위도·경도 지오 프레임 기준으로 다시 검수하는 패치다.

## 포함 사항

- 권역별 지오 프레임 기반 마커 재투영
- 마커 상세 패널에 위도·경도, 투영좌표, 좌표 상태 표시
- 검수 모드에서 GEO PROJECTION QA 패널 표시
- 세계 / 동아시아 / 유럽 / 북미 / 남아시아 / 동남아·인도양 / 오세아니아 / 중동 / 아프리카 권역 상태 표시
- 대륙 잘림 완화를 위한 지도 프레임 보강
- 드래그·줌 중 검은 빈공간이 보이지 않도록 같은 지도 이미지를 저명도 연장 대륙 배경으로 표시
- 실제 마커, 오염권, 봉쇄선은 주 지도 관제면 기준으로만 렌더링

## 연장 대륙 배경 처리

빈공간을 검은색으로 막지 않고, 같은 지도 이미지를 확대·저명도 처리한 `pc553-bleed-map` 레이어를 주 지도 뒤에 깔았다. 이 레이어는 시각적 연장 배경일 뿐이며, 마커 좌표계에는 관여하지 않는다.

## 건드리지 않은 것

- 기록보관서 연출
- F.H.C 검열 연출
- 영상 기록 피드
- 개체 스캔 연출
- 세력관계도 구조
- 전체 효과음 믹싱

## 다음 검수 포인트

- 세계지도에서 대륙이 잘려 보이는지
- 각 권역에서 마커가 실제 지명 근처에 있는지
- 아프리카/중동/오세아니아 권역에서 표면 판정이 맞는지
- 드래그 시 검은 빈공간 대신 연장 대륙 배경이 자연스럽게 보이는지
- 검수 모드에서 위도·경도 / 투영좌표 / 좌표상태가 보이는지
