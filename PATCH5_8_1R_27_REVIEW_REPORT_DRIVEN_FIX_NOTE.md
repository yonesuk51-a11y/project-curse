# PATCH 5.8.1R-27 — ReviewReportDrivenFix

## 목적

R26 자체 검수 리포트에서 드러난 관계권 불균형을 기준으로, 지역지도 데이터의 봉쇄선·봉쇄 거점 짝과 단독 존 관계권을 정리했다. R24 이후의 새 단일 지도 모듈 구조는 유지하며, 기존 누적 지도 패치 방식으로 되돌리지 않는다.

## 핵심 수정

- 봉쇄선만 있고 봉쇄 거점이 없던 관계권에 해상/항만/내륙 봉쇄 거점을 추가했다.
- 봉쇄 거점만 있고 봉쇄선이 없던 관계권에 짧은 항만/검문 차단선을 추가했다.
- 단독 후방권/완충권에는 후방 통신소, 기록 관제소, 감시소, 회수소 등을 보강했다.
- 레드존/블랙존 단독 관계권은 계속 경고하되, 그린존/화이트존/일부 후방 운영권 단독은 자동 점검 경고로 과하게 잡지 않도록 검수 규칙을 완화했다.
