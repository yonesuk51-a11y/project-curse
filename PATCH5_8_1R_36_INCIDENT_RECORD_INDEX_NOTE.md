# PATCH 5.8.1R-36 — IncidentRecordIndex

## 목적

R36은 지역지도 안의 사건 좌표, 현상 기록, 봉쇄선, 봉쇄 거점, 작전 시설, 통신 이상, 관계권을 목록으로도 찾을 수 있게 만드는 색인 패치다.

R35까지 지도↔문서 이동 흐름이 정리되었으므로, R36에서는 지도 요소가 늘어났을 때 사용자가 원하는 기록 좌표를 빠르게 찾는 데 집중한다.

## 추가 내용

### 1. RECORD INDEX 패널

우측 정보 패널에 `RECORD INDEX` 블록을 추가했다.

색인 카테고리:

- 필터 연동
- 전체 색인
- 오염 구역
- 사건 좌표
- 현상 기록
- 봉쇄선
- 봉쇄 거점
- 작전 시설
- 통신 이상
- 관계권

### 2. 필터 연동

`필터 연동` 상태에서는 현재 지도 필터에 맞춰 색인 목록이 자동으로 바뀐다.

예:

- 필터가 `사건 좌표`이면 색인도 사건 좌표 중심으로 표시
- 필터가 `봉쇄선`이면 색인도 봉쇄선 중심으로 표시
- 필터가 `레드존/옐로우존/화이트존` 계열이면 색인은 오염 구역 중심으로 표시

### 3. 검색 기능

색인 검색창을 추가했다.

검색 대상:

- 요소 이름
- 내부 ID
- 권역명
- 관계권 이름
- 역할명
- 상태 설명
- 연결 기록 문구

### 4. 색인에서 지도 요소 바로 열기

색인 항목을 누르면 해당 권역으로 이동하고 해당 요소를 선택한다.

- 세계지도에서 눌러도 해당 상세 권역으로 이동
- 상세지도에서 눌러도 해당 요소 선택
- 선택 후 우측 패널에 기존 관련 기록/이동 흐름 버튼 유지

### 5. 관계권 색인

`관계권` 색인을 추가했다.

관계권 색인에서는 같은 작전권/사건권/봉쇄권에 속한 요소 수와 주요 유형을 표시하고, 선택 시 대표 요소를 열어 같은 관계권이 강조되도록 했다.

### 6. 검수 모드 연동

검수 모드가 켜져 있으면 색인 목록에도 내부 지도 요소 ID가 표시된다.

일반 모드에서는 내부 ID를 숨긴다.

### 7. SELF REVIEW 보강

자체 검수 리포트에 아래 항목을 추가했다.

- 현재 색인 종류
- 검색어
- 색인 항목 수

## 유지 사항

- R31 지도→문서 연결 유지
- R32 겹친 요소 후보 선택 메뉴 유지
- R33 문서→지도 역방향 연결 유지
- R34 링크 무결성 검수 유지
- R35 지도↔문서 복귀 흐름 유지
- 세계지도 권역 선택 전용 상태 유지
- 새 존 체계 추가 없음
- QA 폴더 복구 없음
