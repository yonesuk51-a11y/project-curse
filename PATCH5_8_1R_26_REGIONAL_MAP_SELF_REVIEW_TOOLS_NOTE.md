# PATCH 5.8.1R-26 — RegionalMapSelfReviewTools

## 목적

스크린샷 업로드를 많이 하지 않아도 지역지도 상태를 텍스트로 검수할 수 있도록, R24 재작성 지도 모듈에 자체 검수 리포트 도구를 추가한다.

## 적용 내용

- 우측 정보 패널에 `SELF REVIEW` 블록 추가.
- 현재 권역, 필터, 표시/전체 요소 수, 존/선 개수, 선택 요소 ID 표시.
- `검수 리포트 생성` 버튼 추가.
- `리포트 복사` 버튼 추가.
- 리포트 내용:
  - 지도 모듈 버전
  - 현재 권역
  - 현재 필터
  - 검수 모드 ON/OFF
  - 선택 요소 정보
  - 전체/표시/숨김 요소 수
  - 유형별 표시 요소 수
  - 존 색상별 표시 요소 수
  - 관계권 그룹 목록
  - 중복 ID 및 필드 누락 자동 점검
  - 검수 포인트 안내
- 세계지도에서는 데이터 레이어 없음, 권역 버튼 수, 버튼 위치/크롭 검수 포인트를 출력.
- 일반 화면의 코드 라벨 비노출 및 검수 모드의 코드 라벨 표시 구조는 유지.

## 유지 사항

- 세계지도는 권역 선택 화면으로 유지.
- 세계지도에는 존/시설/사건/봉쇄선 표시 없음.
- R25의 곡선형 봉쇄선, 관계권 강조, 존 톤 보정 유지.
- 필터 목록과 동작 기준 유지.
- QA 폴더 복구 없음.

## 검수 방식

이후 문제 보고 시 스크린샷 대신 아래 형식의 리포트 텍스트를 복사해 전달할 수 있다.

```text
[U.A.C 지역지도 자체 검수 리포트]
버전: ...
권역: ...
필터: ...
요소 수: ...
관계권: ...
자동 점검: ...
```
