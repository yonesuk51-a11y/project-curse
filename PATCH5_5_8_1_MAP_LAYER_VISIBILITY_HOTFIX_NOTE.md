# PATCH 5.5.8.1 — MapLayerVisibilityHotfix

## 기준 버전
- 기준: `MapPatch5_5_8_MapLayerDeclutterQA`
- 목적: 5.5.7/5.5.8에서 추가된 전술 시각 레이어가 ZOOM 4~5에서 여전히 겹쳐 보이는 문제를 좁은 범위로 완화한다.

## 핵심 수정

### 1. ZOOM 5 무선택 상태 강제 요약화
- 선택 마커가 없을 때 ZOOM 5에서 반경/링크/미니 카드/시설 셀을 숨긴다.
- 해칭은 배경 수준의 낮은 명도만 유지한다.
- ZOOM 5는 “전체 정보 표시”가 아니라 “선택 대상 분석” 단계로 고정한다.

### 2. 선택 마커 중심 포커스 강화
- 마커 선택 시 해당 코드와 연결된 반경/링크/미니 카드만 보이게 한다.
- 무관한 링크, 카드, 시설 셀은 숨김 처리한다.
- 무관한 해칭은 배경 명도까지 낮춘다.

### 3. 보조/검수 라벨 잔류 차단
- 일반 모드에서 `HELPER LAYER`, `FAC-*`, 시설 태그, 보조 분석 라벨이 남는 문제를 더 강하게 차단한다.
- 검수 모드에서는 기존 QA 패널을 유지한다.

### 4. FEEDS 패널 축소
- FEEDS 패널의 투명도와 폭을 낮춰 지도 본문을 덜 가리게 한다.
- 작은 화면에서는 FEEDS/범례를 숨긴다.
- 4개 이상 피드는 기본 표시에서 숨겨 패널 밀도를 줄인다.

## 건드리지 않은 것
- 지도 줌/드래그 조작 로직 변경 없음
- 좌표 투영 변경 없음
- 마커 데이터 변경 없음
- 세력관계도 변경 없음
- 기록보관서 변경 없음
- F.H.C/영상/개체 특수 연출 변경 없음
- 전체 효과음 믹싱 변경 없음

## 검수 포인트
- ZOOM 5에서 아무 마커도 선택하지 않았을 때 화면이 조용해지는지
- 마커 선택 후 해당 마커 주변 정보만 강조되는지
- 전체 링크선/반경/미니 카드가 동시에 떠서 겹치지 않는지
- 일반 모드에서 `HELPER LAYER`, `FAC-B` 같은 문구가 보이지 않는지
- FEEDS 패널이 지도 주요 영역을 덜 가리는지
