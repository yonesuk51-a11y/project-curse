# PATCH5_8_1R-45 — PrecisionContentLayerPass

## 목적

R44에서 추가된 상세지도 전용 확대/초점 보기 기능을 기반으로, 확대·정밀 보기에서만 드러나는 보조 콘텐츠 레이어를 추가한다.

세계지도는 계속 권역 선택 전용 화면으로 유지하며, 줌/드래그/FIT/WIDE 기능을 추가하지 않는다.

## 핵심 변경

- 상세 보기 단계를 `요약 / 기본 / 확대 / 정밀`로 확장
- 기본 화면에는 기존 핵심 지도 요소만 유지
- 확대 보기에서 보조 시설·통신점·검문 보조점 일부 표시
- 정밀 보기에서 관측 실패점·회수 실패 좌표·부표 신호 누락점·보조 검문 게이트 표시
- 전체 지도 요소 수를 149개에서 181개로 확장
- 추가 콘텐츠는 `assets/data/uac_region_map_data.js` 데이터 모듈에만 반영
- 렌더러 본체에는 보기 단계별 표시 규칙만 추가

## 보기 단계 기준

- `요약`: 핵심 존, 주요 봉쇄선, 대표 사건·시설 위주 표시
- `기본`: R44 기준 표준 작전도
- `확대`: 보조 관측소, 통신 중계점, 항만/해상 보조점 표시
- `정밀`: 관측 실패 좌표, 회수 실패 지점, 검문 실패 지점, 보조 부표 표시

## 권역별 추가 수

각 상세 권역에 보조 콘텐츠 4개씩 추가했다.

- 동아시아: 4개
- 유럽: 4개
- 북미: 4개
- 남아시아: 4개
- 동남아·인도양: 4개
- 오세아니아: 4개
- 중동: 4개
- 아프리카: 4개
