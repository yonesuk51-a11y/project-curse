# MapPatch 5.15.2as — RecordSequence_StageViewerInspired

기준: 5.15.2ar FrameAudio_RebaseSafe.

## 목적

F.H.C 극비 보안 문서와 불멸을 향하여의 특수 기록 시퀀스에만 영상 참고 기반의 화면 문법을 제한 적용했다. 네이티브 문서 본문 레이아웃은 변경하지 않는다.

## 반영

- F.H.C 시퀀스: warning/title, two-column briefing, evidence-photo, warning/end card 레이아웃 클래스를 자동 적용.
- Immortality 시퀀스: terminal brief, terminal log, field photo, personnel 레이아웃 클래스를 자동 적용.
- 증거 사진형 F.H.C 페이지 일부는 이미지 우선/캡션/본문 구조로 표시.
- 모바일에서는 모든 시퀀스를 1열로 접어 이미지와 텍스트가 겹치지 않게 처리.
- 기존 5.15.2ap 문서 흐름 및 5.15.2ar 프레임/오디오 안정화 유지.

## 보류

- 원본 참고 영상 파일 자체 삽입 없음.
- 일반 문서 페이지의 이미지/텍스트 배치 변경 없음.
- 관계도/지도/기록보관소 구조 변경 없음.
