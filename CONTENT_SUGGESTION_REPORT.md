# Content Suggestion Report / MapPatch6.2.8

이 리포트는 새 설정을 자동으로 확정하지 않고, 추가하면 좋아질 후보만 제안한다.

## Summary

### 세계 개요 — 현재 통제 상태 요약 카드 후보
- Priority: medium
- Why: 첫 화면에서 세계 상태를 빠르게 파악할 수 있음
- Candidate: 오염 확산 상태, 주요 봉쇄선 상태, 기록 무결성, 최근 작전 로그 요약 카드
- Guard: 사용자 승인 전 핵심 설정으로 확정하지 않음

### 세력 — 세력별 관련 기록/지도 링크 후보
- Priority: high
- Why: 세력기록, 기록보관서, 지역지도가 따로 노는 느낌을 줄임
- Candidate: 각 세력 카드에 관련 기록 문서와 관련 작전구역도 링크 후보 추가
- Guard: 사용자 승인 전 핵심 설정으로 확정하지 않음

### 지역지도 — 작전구역-기록보관서 연결 후보
- Priority: high
- Why: 반추상 작전구역도의 서사적 연결성을 강화함
- Candidate: 통신 이상권 ↔ Ghost Channel 기록, 항만 화이트존 ↔ 검문소 기록, 레드존 ↔ 봉쇄/관측 기록 연결 후보
- Guard: 사용자 승인 전 핵심 설정으로 확정하지 않음

### 기록보관서 — 문서 내부 관련 좌표/세력 섹션 후보
- Priority: high
- Why: 설정글이 지도/세력 UI와 연결되어 보임
- Candidate: 각 문서 하단에 관련 세력, 관련 작전구역, 관련 사건 기록 후보 섹션
- Guard: 사용자 승인 전 핵심 설정으로 확정하지 않음

### 오디오/연출 — 제한 기록 오류음 후보
- Priority: low
- Why: 잠긴 문서가 생길 경우 조작 피드백이 명확해짐
- Candidate: 낮은 구형 컴퓨터 오류음 후보. 현 단계에서는 자동 추가하지 않음
- Guard: 사용자 승인 전 핵심 설정으로 확정하지 않음

### 영상 기록 — VHS 전용 효과음 후보
- Priority: low
- Why: 일반 UI와 분리된 복구 영상 기록이 생길 때 분위기 강화 가능
- Candidate: VHS play/stop 효과음은 영상 기록 전용 후보로만 유지
- Guard: 사용자 승인 전 핵심 설정으로 확정하지 않음

## Auto-add 제한

- 새 세력, 새 존 체계, 새 대형 사건, 새 개체 분류 체계, 기존 핵심 명칭 변경은 사용자 승인 전 자동 추가하지 않는다.
