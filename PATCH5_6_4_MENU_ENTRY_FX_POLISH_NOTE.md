# PATCH 5.6.4 — MenuEntryFXPolish + FactionRelationLayoutFix

## 기준 버전
- MapPatch5_6_3_FactionRelationVisualUpgrade

## 목적
원래 다음 순서였던 메뉴 진입 연출 정리 패치에 5.6.3 세력관계도 레이아웃 보정 항목을 함께 포함한다.

## 포함 항목

### 1. 메뉴별 진입 연출 정리
왼쪽 메뉴 진입 시 동일한 로딩 문구가 반복되지 않도록 메뉴별 전용 진입 HUD를 추가했다.

- 세계 역사: CHRONICLE INDEX REBUILD
- 세력 정보: FACTION DOSSIER ACCESS
- 세력 관계: RELATION TRACE MATRIX
- 지역 지도: REGIONAL SURFACE LINK
- 기록 보관소: ARCHIVE SURFACE OPEN

### 2. 세력관계도 레이아웃 보정
5.6.3에서 U.A.C 중심 구조는 들어갔지만 방사형 느낌이 약하고 선 교차가 많았던 문제를 보정했다.

- U.A.C 중심 방사형 링 구조 재배치
- 1차 연결 / 후속 지원 / 외곽 위험 세력 간격 재조정
- 관계선을 직선 위주에서 곡선 경로로 조정
- 선택 노드 중심 포커스 강화
- 비선택 노드와 선의 흐림 처리 개선

### 3. 관계 분석 패널 보강
선택 세력 패널에 분석 항목을 추가했다.

- 선택 세력
- 상태 태그
- 관계 등급
- 감시 상태
- 직접 연결
- 위험도
- 관련 기록
- U.A.C 판단

### 4. C.P.D 표기 정리
C.P.D는 조직 단위로만 유지하고, C.P.D 대피버스·차량·장비·지도 마커는 세력관계도 노드에서 제외한다는 문구를 명시했다.

### 5. 오디오 범위
노드 클릭 효과음은 임시 유지했다. 전체 오디오 믹싱은 5.7 AudioFXPolish에서 정리한다.

## 건드리지 않은 것
- 지역지도 줌/좌표/레이어 로직
- 기록보관서 구조
- F.H.C / 영상 / 개체 특수 연출
- 전체 효과음 믹싱
