# PATCH 5.5.4 — MapViewportMarkerStackQA

## 기준

- 기준 패치: `MapPatch5_5_3_MapProjectionMarkerRebuild`
- 목적: 지도 좌표계 재구축 이후 남은 뷰포트/마커 선택성 문제 보정

## 적용 내용

### 1. 마커 중첩 감지

- 현재 권역의 `[data-map-item]`을 투영 좌표 기준으로 검사합니다.
- 현재 줌 배율에 따라 중첩 판정 범위를 줄입니다.
- 같은 좌표권에 2개 이상 표식이 있으면 `+N` 클러스터 배지를 생성합니다.

### 2. 중첩 마커 선택 목록

- 중첩된 마커를 클릭하면 바로 하나만 열지 않고 `MARKER STACK DETECTED` 목록을 표시합니다.
- 목록에서 선택한 항목이 기존 마커 상세 패널로 연결됩니다.
- 상세 패널에는 `중첩 상태`가 추가됩니다.

### 3. 클릭 판정 분리

- 보이는 마커와 클릭 hit-area를 분리했습니다.
- facility / anomaly / incident / blockade-node 계열에 투명 클릭 원을 추가해 작은 표식도 누르기 쉽게 했습니다.

### 4. 검수 모드 보강

- `MARKER STACK QA` 패널을 추가했습니다.
- 현재 권역, 줌 단계, 중첩 그룹 수, 선택 모드가 표시됩니다.

### 5. 뷰포트 보정

- 5.5.3의 연장 대륙 배경을 유지하고 overscan 범위를 넓혔습니다.
- 검은 빈공간 노출 체감을 줄이되, 실제 마커 좌표는 주 지도면 기준으로 유지합니다.

### 6. 가벼운 LOD 보강

- ZOOM 4~5에서 도로망/시설 블록 느낌의 가벼운 시각 레이어가 표시됩니다.
- 실제 좌표 데이터나 기존 마커 데이터는 변경하지 않습니다.
