# PATCH 5.8.1R-48 — DetailMapPanAndWheelZoomControlPass

## 목적
R47에서 재구성한 `줌 1 ~ 줌 5` 단계형 전술 표시를 유지하면서, 상세 권역지도에 한정해 마우스 휠 줌과 제한 드래그 이동을 추가한다.

## 적용 범위
- 세계지도: 권역 선택 전용 유지 / 줌 없음 / 드래그 없음 / FIT/WIDE 없음 / 데이터 마커 없음
- 상세지도: 줌 1~줌 5 유지
- 상세지도 줌 2~줌 5: 빈 지도 영역 드래그 이동 가능
- 마우스 휠: 상세지도에서 줌 단계 전환

## 동작 기준
- 줌 1: 전체 작전도 고정, 드래그 이동 없음
- 줌 2~줌 5: 마우스 드래그로 제한 이동 가능
- 마우스 휠 위: 줌 단계 증가
- 마우스 휠 아래: 줌 단계 감소
- 세계지도에서는 휠/드래그 조작 무시
- 선택 중심/관계권 중심 보기와 드래그 위치는 함께 사용 가능
- `줌/위치 초기화`로 줌 1과 기본 위치로 복귀

## 유지
- R47 줌 단계별 점 표시/상세 표시 구조 유지
- R31~R43 지도↔문서 연결 유지
- R42 RECORD INDEX 성능 정리 유지
- R43 시각 핫픽스 유지
- 일반 화면 내부 ID 비노출 유지
- 검수 모드 내부 ID 표시 유지
