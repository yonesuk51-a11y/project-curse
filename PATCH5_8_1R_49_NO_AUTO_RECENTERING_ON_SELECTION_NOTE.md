# PATCH5_8_1R_49 — NoAutoRecenteringOnSelection

## 목적

R48에서 상세지도 줌 2~5 드래그 이동과 마우스 휠 줌이 추가된 뒤, 시설/마커/존/봉쇄선 등을 선택할 때 선택 요소 중심으로 자동 복귀되는 흐름을 제거한다.

## 변경 사항

- 일반 지도 요소 선택 시 자동으로 `선택 중심` 초점으로 전환하지 않도록 수정.
- 점 표시 요소, 상세 아이콘, 존, 봉쇄선, 목록 항목, 겹친 후보 메뉴 선택 시 현재 pan 위치를 유지.
- 이미 드래그해서 맞춰둔 화면 위치가 일반 선택만으로 초기화되지 않도록 보정.
- `선택 중심` / `관계권 중심` 버튼을 직접 누른 경우에만 중심 초점 이동이 발생하도록 분리.
- `줌/위치 초기화` 버튼은 기존처럼 줌 1과 기본 위치로 복귀.
- 색인 또는 문서 연결로 특정 지도 요소를 직접 여는 경우에는 필요한 최소 이동을 유지.
- SELF REVIEW/API용 선택 위치 유지 검수 함수 추가.

## 유지 사항

- 세계지도는 권역 선택 전용이며 줌/드래그/FIT/WIDE 없음.
- 상세지도 마우스 휠 줌 1~5 유지.
- 줌 2~5 드래그 이동 유지.
- R47 줌 단계별 점 표시/상세 표시 구조 유지.
- RECORD INDEX, LINK AUDIT, SELF REVIEW, 지도↔문서 연결 유지.
- 일반 화면 내부 ID 비노출 유지.
- 검수 모드 내부 ID 표시 유지.
